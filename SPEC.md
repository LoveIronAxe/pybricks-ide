# Pybricks 机器车 IDE — 页面规格说明书

> 本文档整理皇上口述的 GUI 需求，作为 AI 编写页面的唯一依据。
> 包含四大页面：车辆 / 地图 / 程序 / 调试

---

## 一、整体架构

### 1.1 布局结构

```
┌────────┬─────────────────────┬──────────────────────────────┐
│  侧边栏 │    列表栏（左小栏）    │       详情栏（最大）          │
│  56px   │    240px ~ 280px     │       flex: 1               │
│  图标+   │  标题 + [+] 横线     │   具体页面内容                 │
│  一两个字  │  可滚动列表           │   根据选中项动态切换           │
└────────┴─────────────────────┴──────────────────────────────┘
```

右上角固定位置：📥 **一键导出Python** 按钮（所有页面可见）

### 1.2 侧边栏（主导航）

- 宽度：56px（仅容图标 + 1-2 个汉字）
- 顶部 → 底部排列（顺序固定）：
  1. 🛒 **车辆**（默认激活）
  2. 🗺️ **地图**
  3. 📄 **程序**
  4. 🔧 **调试**
- 点击切换：整个右侧两栏内容随之切换
- 激活态：左侧有 3px 蓝色指示条，文字高亮

### 1.3 设计风格

- **设计语言**：Material Design 3（参考 design/Material_Design_全套资料/ 目录）
- **主色调**：`#0071e3`（Apple Blue / MD3 Blue 40）
- **弹窗规范**：
  - 背景暗化遮罩（`rgba(0,0,0,0.4)` + `backdrop-filter: blur(4px)`）
  - 弹窗居中，圆角 28px
  - 右上角关闭 / 取消
  - 按钮：取消（灰色 Outlined）+ 创建（蓝色 Filled）

## 二、车辆页面

### 2.1 列表栏

- 顶部：标题"车辆" + 右侧 [+] 加号按钮
- 横线分隔标题与列表
- 列表项：车辆名称，每次点击高亮选中
- 选中后在详情栏展示配置卡片

### 2.2 添加车辆弹窗

**触发**：点击 [+] 按钮

**内容**：
1. **Name** — 文本输入框（车辆名称）
2. **Hub 选择** — 下拉选项，当前固定为 `Prime Hub`
3. **取消键**（灰色）+ **创建键**（蓝色 #0071e3）
4. 点击背景或取消均可关闭弹窗

### 2.3 拼搭俯视图（详情栏核心）

**顶部：拼搭俯视图区域**
- 显示 Prime Hub 主体（102×82mm 比例，CSS 复刻）
- 6个端口（A-F）：俯视图定位，凸出小长方形
- 5×5 点阵显示屏（居中，橙红色发光）
- 4个按钮：参考实物排列
- 接口可点击，点击弹出「连接设备」弹窗

**连接设备弹窗**：
- 大标题：端口号（如 "Port A"）
- 选项列表：从官方文档获取的可用设备中选择
- 取消/确认按钮
- 确认后：接口旁边出现连接短线，指向设备卡片

**设备卡片（跟随 Hub 端口连线）**：
- 稍大长方形
- 左侧 emoji（对应设备类型）+ 右侧名称
- 右上角 ⨉ 圆角叉号（可移除）
- 端口标注在卡片上
- 点击设备卡片可重新编辑

**可用设备（来自 pybricks_docs/ 官方文档）**：
| 设备 | Emoji | 对应端口 |
|------|-------|---------|
| 中型电机 | ⚙️ | A/B/C/D |
| 大型电机 | ⚙️ | A/B/C/D |
| 颜色传感器 | 🔴 | E/F |
| 颜色距离传感器 | 📏 | E/F |
| 力传感器 | 🔲 | A/B/C/D |
| 超声波传感器 | 📡 | E/F |
| 红外传感器 | 📶 | E/F |
| 倾斜传感器 | 📐 | A/B/C/D |
| 遥控器 | 🎮 | — |

### 2.4 爪子/叉子配置（单电机两个状态）

**配置方式**：
- 选中叉子/爪子电机卡片后，详情栏显示该电机的两个可配置状态
- **状态A**（如"张开"）：初始化角度 + 是否启用
- **状态B**（如"闭合"）：初始化角度 + 是否启用
- Checkbox：可选中激活某状态
- 点击按钮可测试爪子开合动作

### 2.5 轮组卡片

**拖拽组成**：
- 从设备列表拖拽两个电机卡片到轮组区域
- 拖拽效果：卡片有明显"被拿走"效果（透明度降低/虚线边框）
- 放置效果：两电机并排，转子朝外，自动标注端口
- 中间有双箭头图标，可翻转左右电机方向

**轮组配置**：
- 轮胎直径（mm）
- 轮胎宽度（mm）
- 左右标识（可翻转）
- axle_track 自动根据轮子直径计算

### 2.6 单光巡线系统卡片

**触发**：将颜色传感器拖入轮组卡片后，标题变为"单光巡线系统"

**配置项**：
- 线颜色（颜色选择器）
- 背景颜色（颜色选择器）
- 传感器在线的哪一侧（左/右单选）
- 反射率阈值（数值输入，范围 0~100）
- 黑白反射率数值（自动计算 threshold）
- **PID 参数**：
  - Kp：数值输入，范围 0.3~4.0，step 0.1
  - 扫描频率（wait_ms）：5~50ms，建议 5~20
- 频率警告：
  - `>30ms`：橙色提示文字"太高了，机器会反应不过来"
  - `<5ms`：红色提示文字"太低可能导致传感器采样不稳定"
- 颜色反射率（背景/线条各一个数值）

## 三、地图页面

### 3.1 列表栏

- 顶部：标题"地图" + [+] 按钮
- 横线分隔
- 列表项：缩略图（64×64px）+ 地图名称
- 支持拖拽文件上传（图片拖入列表栏触发选择）

### 3.2 添加地图弹窗

**触发**：点击 [+] 或拖拽文件

**内容**：
1. 图片预览区（居中）
2. 图片详细信息（分辨率、色彩深度、文件大小）
3. 标题输入框
4. 取消/导入按钮

### 3.3 地图编辑器（详情栏）

**顶部工具栏**：
- 工具按钮组：选择工具 / 标尺工具 / 划线工具 / 点到点测量工具 / 障碍物绘制 / 放置点工具
- 每个工具点选后高亮激活

**地图显示区**：
- 地图图片占满整个宽度（自适应缩放）
- 支持缩放（滚轮）和平移（拖拽）
- 网格叠加显示（可选）

**标尺工具**：
- 在地图上放置带刻度的标尺（1cm 刻度）
- 可旋转、缩放
- 实时显示像素长度

**点到点测量工具**：
- 鼠标点击两次放置两个测量点
- 两点间显示连线
- 完成后弹出输入框：输入实际距离（mm）
- 自动计算像素↔毫米换算系数

**障碍物绘制**：
- 工具栏选择"障碍物绘制"后，可绘制正方形/长方形区域
- 属性面板：设置是否"不可通行" / "可推动"
- 可设置颜色（红色=不可通行，黄色=可推动）
- 障碍物可拖拽移动位置

**搬运物绘制**：
- 可绘制需要搬运的物品
- 设置：颜色要求、形状（方形/圆形）
- 放置后随机位置显示（调试时用）

**巡线路径绘制**：
- 使用划线工具在地图上描绘巡线路线
- 线条颜色可选
- 绘制完成后可保存为"路径"配置

### 3.4 分屏功能

- 地图编辑器可选择分屏：上下分屏 / 左右分屏
- 左侧/上侧：地图编辑
- 右侧/下侧：任务规划视图
- 分屏比例可拖拽调整

- 节点可拖拽位置

- 搬运物可拖动（模拟搬运）

### 4.8 一键导出Python

- 工具栏或右上角按钮
- 将所有代码块/流程图生成为完整 Python 文件
- 自动包含：import 语句、端口常量、init()、calibrate()、follow_line()、turn()、用户程序
- 触发下载 .py 文件

## 七、技术要求

### 7.1 关键技术

- **无路由跳转**：单页应用，所有切换通过 DOM 显示/隐藏
- **无 URL 参数**：状态全存在 localStorage
- **拖拽**：HTML5 Drag & Drop API
- **弹窗**：原生 dialog 元素或遮罩实现
- **响应式**：CSS Flexbox + Media Query
- **物理模拟**：Canvas 2D 绘制 + 简化物理引擎（碰撞检测、差速转向）
- **流程图**：支持节点拖拽、连线、并行分支（限制并发数）
- **图表**：Canvas 绘制实时曲线（反射率/速度）
- **导出**：Blob + URL.createObjectURL 触发 .py 下载

### 7.2 模拟器技术要求

- 物理引擎：简化版（无重力，2D平面）
- 碰撞检测：AABB（轴对齐边界框）
- 转向模拟：根据 axle_track 和轮子直径计算实际转向半径
- 传感器模拟：根据车辆位置计算反射率（在线上/线下/边缘）
- 颜色识别：根据车辆位置判断是否在有色物体上方

### 7.3 代码块拖拽系统

- 每个代码块是独立的 DOM 元素
- 拖拽时生成 Ghost 元素跟随鼠标
- 放置时有视觉反馈（高亮放置区域）
- 代码块可嵌套（if/for 结构内可放子代码块）
- 连接线用 SVG 或 Canvas 绘制

## 九、调试页面（补充细节）

### 9.1 模拟器（全息仿真）

**显示内容**：
- 地图背景（与地图页同步）
- 车辆俯视图（根据车辆配置动态渲染：Hub大小/轮间距/爪子形态）
- 障碍物（红色不可通行 / 黄色可推动）
- 搬运物（带颜色标识）
- 巡线路径（从地图页导入）
- 轨迹显示（执行时实时绘制行驶轨迹）

**动作模拟**：
- 轮子旋转动画
- 爪子开合动画（对应两个状态的配置）
- 传感器扫描动画（颜色变化）
- 搬运物被推动/抓取动画
- 到达目标后停止

**物理引擎细节**：
- 碰撞检测：AABB 轴对齐边界框
- 摩擦力：影响停止距离
- 差速转向：根据 axle_track 计算实际转向半径
- 传感器模拟：根据车辆位置计算反射率（线上/线下/边缘）+ 颜色识别

### 9.2 搬运物系统

- 搬运物属性：颜色（红/黄/绿/蓝）、形状（方形/圆形）
- 有些搬运物位置固定，有些随机放置（调试时随机）
- 车辆经过可推动"可推动"障碍物
- 爪子夹取搬运物后移动

### 9.3 控制台输出（每个动作完成后的日志）

- 格式：`[时间戳] [动作名] 参数 → 结果`
- 例如：
  - `[00:01.23] follow_line: distance=300mm, kp=1.5 → 到达目标`
  - `[00:02.45] claw_close: angle=-300 → 完成`
  - `[00:03.12] turn: angle=90° → 完成`
- 颜色区分：成功（绿）/ 警告（黄）/ 错误（红）
- 每行可点击定位到对应流程图节点

### 9.4 实时数据面板

- 电机速度（deg/s）
- 电机当前角度
- 传感器读数（反射率0~100）
- 巡线误差（error = reflection - threshold）
- 车辆坐标（x, y, heading）
- 当前执行节点高亮

### 9.5 传感器实时曲线图

- Canvas 绘制（matplotlib 风格）
- 反射率实时曲线（每秒更新）
- 可缩放/平移
- 阈值线叠加显示

### 9.6 调试控制栏

- 按钮：▶️开始 / ⏸️暂停 / ⏹️停止 / 🔄重置
- 速度倍率：0.5x / 1x / 2x
- 单步执行：逐节点执行
- 当前节点高亮
