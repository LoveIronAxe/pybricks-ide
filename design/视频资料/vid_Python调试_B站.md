## 🔑 关键设计启示

PyBricks IDE 调试功能设计建议：

1. **参考 PyCharm 三区布局**：断点栏（左侧）+ 变量面板（下侧/右侧）+ 调用栈（右侧），简洁高效
2. **集成 Web Debugger**：基于 Chrome DevTools Protocol（CDP）或 Python `debugpy` 实现 Web 版调试器，与 VS Code 同款架构
3. **分步执行可视化**：参考 Thonny，将"当前执行行高亮+变量实时变化"作为核心体验
4. **积木级调试**：创新点——支持按"积木块"为单位进行分步执行，而非传统逐行，适合青少儿理解程序流程
