# 动效与过渡系统

## 1. MD3 动效层级

MD3（Material Design 3）将动画时长分为三个层级：

| 层级 | 时长 | CSS 变量 | 适用场景 |
|------|------|---------|---------|
| 短（Short） | 100–200ms | `--md-motion-duration-short` | 交互反馈、快速切换 |
| 中（Medium） | 250–400ms | `--md-motion-duration-medium` | 组件展开、弹窗出现 |
| 长（Long） | 400–800ms | `--md-motion-duration-long` | 页面切换、大型布局变化 |

```css
/* MD3 动效时长变量 */
:root {
  --md-motion-duration-short:   150ms;  /* 快速反馈 */
  --md-motion-duration-medium:   300ms;  /* 标准过渡 */
  --md-motion-duration-long:      500ms;  /* 大型动画 */

  /* 超出上述范围的极端值（慎用） */
  --md-motion-duration-enter:     200ms;
  --md-motion-duration-exit:      150ms;  /* 退出通常比进入快 */
}
```

## 2. 缓动曲线（Easing）规范

### 2.1 标准缓动曲线

| 曲线名称 | CSS 值 | 适用场景 |
|---------|--------|---------|
| 标准（Standard） | `cubic-bezier(0.2, 0, 0, 1)` | 大多数组件的移动 |
| 强调（Emphasized） | `cubic-bezier(0.2, 0, 0, 1)` | FAB、弹窗（稍快） |
| 减速（Decelerated） | `cubic-bezier(0, 0, 0, 1)` | 进入动画 |
| 加速（Accelerated） | `cubic-bezier(0.3, 0, 1, 1)` | 退出动画 |
| 线性（Linear） | `linear` | 颜色/透明度变化（有时） |

```css
:root {
  /* 标准曲线 */
  --md-motion-easing-standard:  cubic-bezier(0.2, 0, 0, 1);
  --md-motion-easing-emphasized: cubic-bezier(0.2, 0, 0, 1); /* 同标准，但配合更长时长 */

  /* 进出曲线（入场慢出快 vs 进场快出场慢） */
  --md-motion-easing-enter: cubic-bezier(0, 0, 0, 1);   /* 减速入场 */
  --md-motion-easing-exit:  cubic-bezier(0.3, 0, 1, 1); /* 加速退场 */
}
```

### 2.2 缓动曲线可视化理解

```
标准曲线: 0.2, 0, 0, 1
  - 动画开始较慢（0→0.2 段）
  - 动画中段加速
  - 动画结束前减速（接近 1 时变慢）
  - 适合元素的位移/展开

减速曲线: 0, 0, 0, 1
  - 从最高速度开始
  - 逐渐减速直到停止
  - 适合"出现"类动画（元素突然进入视野）

加速曲线: 0.3, 0, 1, 1
  - 从静止开始
  - 逐渐加速
  - 突然结束
  - 适合"消失"类动画（元素突然离开）
```

## 3. 组件进入/退出动画

### 3.1 淡入淡出（Fade）

```css
/* 淡入 */
@keyframes fade-in {
  from { opacity: 0; }
  to   { opacity: 1; }
}

/* 淡出 */
@keyframes fade-out {
  from { opacity: 1; }
  to   { opacity: 0; }
}

.fade-enter {
  animation: fade-in var(--md-motion-duration-short) var(--md-motion-easing-enter) both;
}

.fade-exit {
  animation: fade-out var(--md-motion-duration-short) var(--md-motion-easing-exit) both;
}
```

## 4. 🔴 P1 重点：弹窗缩放动画

### 4.1 弹窗进入动画（Scale + Fade）

```css
/* 🔴 弹窗进入：从小到大缩放 + 淡入 */
@keyframes dialog-enter {
  from {
    opacity: 0;
    transform: scale(0.9); /* 从 90% 开始 */
  }
  to {
    opacity: 1;
    transform: scale(1);   /* 到 100% 结束 */
  }
}

/* 🔴 弹窗退出：缩回到小 + 淡出 */
@keyframes dialog-exit {
  from {
    opacity: 1;
    transform: scale(1);
  }
  to {
    opacity: 0;
    transform: scale(0.95); /* 缩小到 95%（比进入时缩小幅度小，更自然） */
  }
}

.dialog-backdrop {
  position: fixed;
  inset: 0;
  background: rgba(0, 0, 0, 0.32); /* MD3 scrim */
  display: flex;
  align-items: center;
  justify-content: center;
  z-index: 1000;

  /* 进入/退出的背景动画 */
  animation: fade-in var(--md-motion-duration-short) var(--md-motion-easing-enter) both;
}

.dialog-backdrop.closing {
  animation: fade-out var(--md-motion-duration-short) var(--md-motion-easing-exit) both;
}

.dialog-surface {
  background: #fff;
  border-radius: 28px; /* MD3 大圆角 */
  padding: 24px;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.24);
  min-width: 280px;
  max-width: 560px;
  max-height: 85vh;
  overflow: auto;

  /* 🔴 弹窗核心缩放动画 */
  animation: dialog-enter 300ms cubic-bezier(0, 0, 0, 1) both;
}

.dialog-surface.closing {
  animation: dialog-exit 200ms cubic-bezier(0.3, 0, 1, 1) both;
}
```

### 4.2 弹窗 HTML 结构

```html
<!-- 弹窗容器 -->
<div class="dialog-backdrop" id="myDialog">
  <div class="dialog-surface" role="dialog" aria-modal="true">
    <h2>弹窗标题</h2>
    <p>弹窗内容...</p>
    <button onclick="closeDialog()">关闭</button>
  </div>
</div>
```

### 4.3 弹窗 JavaScript 控制

```javascript
// 打开弹窗
function openDialog() {
  const backdrop = document.getElementById('myDialog');
  const surface = backdrop.querySelector('.dialog-surface');

  // 重置动画状态
  surface.classList.remove('closing');
  backdrop.classList.remove('closing');

  // 显示弹窗
  backdrop.style.display = 'flex';

  // 强制重排后重新触发动画
  surface.offsetHeight; // 或 requestAnimationFrame
}

// 关闭弹窗
function closeDialog() {
  const backdrop = document.getElementById('myDialog');
  const surface = backdrop.querySelector('.dialog-surface');

  // 添加退出动画
  surface.classList.add('closing');
  backdrop.classList.add('closing');

  // 动画结束后隐藏
  surface.addEventListener('animationend', () => {
    backdrop.style.display = 'none';
    surface.classList.remove('closing');
    backdrop.classList.remove('closing');
  }, { once: true });
}
```

### 4.4 弹窗动画参数总结

| 参数 | 进入（Enter） | 退出（Exit） |
|------|-------------|-------------|
| 缩放起始 | `scale(0.9)` | `scale(1)` |
| 缩放结束 | `scale(1)` | `scale(0.95)` |
| 透明度 | `opacity: 0 → 1` | `opacity: 1 → 0` |
| 时长 | 300ms | 200ms（退出稍快） |
| 缓动（缩放） | `cubic-bezier(0, 0, 0, 1)` 减速 | `cubic-bezier(0.3, 0, 1, 1)` 加速 |
| 缓动（背景） | `cubic-bezier(0, 0, 0, 1)` | `cubic-bezier(0.3, 0, 1, 1)` |

### 5.1 列表淡入效果

```css
/* 🔴 列表项淡入：从下方滑入 + 淡入，依次延迟 */
@keyframes list-item-enter {
  from {
    opacity: 0;
    transform: translateY(12px); /* 从下方 12px 进入 */
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 🔴 单个列表项基础样式 */
.list-item {
  opacity: 0; /* 初始隐藏，等待动画 */
  animation: list-item-enter 200ms cubic-bezier(0, 0, 0, 1) both;

/* 🔴 依次延迟（stagger effect），每个 item 延迟 30-50ms */
.list-item:nth-child(1)  { animation-delay: 0ms; }
.list-item:nth-child(2)  { animation-delay: 30ms; }
.list-item:nth-child(3)  { animation-delay: 60ms; }
.list-item:nth-child(4)  { animation-delay: 90ms; }
.list-item:nth-child(5)  { animation-delay: 120ms; }
/* ... 以此类推，最多覆盖前 10-15 项 */
```

## 8. prefers-reduced-motion 无障碍支持

```css
/* 减少动画偏好：无动画 / 瞬时切换 */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    /* ⚠️ 使用 0.01ms 而非 0，避免某些浏览器认为 duration: 0 会禁用 transition */
  }
}
```

```javascript
// JavaScript 检测减少动画偏好
const prefersReducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches;

function animateElement(element) {
  if (prefersReducedMotion) {
    // 无动画，直接显示最终状态
    element.style.opacity = 1;
    element.style.transform = 'none';
  } else {
    // 正常动画
    element.classList.add('fade-enter');
  }
}
```

```html
<!-- HTML 层面也可使用这个测试脚本快速验证 -->
<script>
  const mq = window.matchMedia('(prefers-reduced-motion: reduce)');
  console.log('用户偏好减少动画:', mq.matches);
</script>
```

## 9. 本项目 IDE 具体应用

### 9.1 弹窗应用场景

| 弹窗类型 | 进入时长 | 退出时长 | 缩放范围 | 备注 |
|---------|---------|---------|---------|------|
| 确认对话框 | 200ms | 150ms | 0.9→1.0 | 小型，简洁 |
| 设备配置弹窗 | 300ms | 200ms | 0.9→1.0 | 中型，含表单 |
| 固件更新弹窗 | 300ms | 250ms | 0.85→1.0 | 大型，含进度条 |

### 9.2 侧边栏应用场景

| 场景 | 动画类型 | 时长 | 缓动 |
|------|---------|------|------|
| 打开侧边栏 | slide-in-left | 250ms | `cubic-bezier(0.2, 0, 0, 1)` |
| 收起侧边栏 | slide-out-left | 200ms | `cubic-bezier(0.3, 0, 1, 1)` |

|------|---------|------|

## 11. 调试与验证

### 11.1 如何验证动画流畅度

1. **Chrome DevTools**：打开 Performance 面板，录制动画帧
2. **FPS 监控**：使用 `chrome://flags/#show-fps-counter` 或 DevTools Rendering 面板
3. **Android 开发者选项**：显示 GPU 渲染配置文件（条形图）
4. **prefers-reduced-motion 测试**：
   - macOS：系统设置 → 辅助功能 → 显示 → 减弱动态效果
   - Windows：设置 → 轻松使用 → 显示 → 在 Windows 中显示动画
   - iOS：设置 → 辅助功能 → 减弱动态效果

### 11.2 常见动画性能问题

| 问题 | 解决方案 |
|------|---------|
| 动画卡顿（掉帧） | 仅使用 `transform` 和 `opacity`，避免触发 layout/paint |
| 动画闪烁 | 添加 `will-change: transform`（但用完要移除）|
| 字体加载跳动 | 使用 `font-display: swap` + `@font-face` |
| 移动端动画慢 | 减少动画时长，降低 `calc()` 复杂度 |
