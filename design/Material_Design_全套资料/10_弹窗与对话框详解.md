# 10_弹窗与对话框详解

对话框用于需要用户立即注意或做出决策的场景，是 IDE 中确认操作、显示详情的重要组件。

---

## 🔴P1 Dialog（对话框）

### 尺寸规范

| 尺寸 | 宽度 | 适用场景 |
|------|------|----------|
| Small | 280px | 简单确认（如删除确认） |
| Medium | 400px | 标准对话框（默认） |
| Large | 560px | 复杂表单或列表 |
| Extra Large | 640px | 非常复杂的内容 |

### 🔴P1 圆角规范

```
Dialog 圆角：28px（关键规范）
容器内元素圆角：12px（子卡片）/ 16px（按钮区域）
```

### CSS 实现

```css
.md3-dialog {
  /* 定位 */
  position: fixed;
  inset: 0;
  z-index: 1000;
  
  /* 居中 */
  display: flex;
  align-items: center;
  justify-content: center;
  
  /* 最小高度 */
  min-height: 100vh;
  
  /* 禁用背景滚动 */
  overflow: hidden;
}

/* 遮罩层（Scrim） */
.md3-dialog__scrim {
  position: absolute;
  inset: 0;
  background-color: rgba(0, 0, 0, 0.32);
  opacity: 0.4;
  transition: opacity 200ms ease;
}

/* 对话框容器 */
.md3-dialog__surface {
  /* 尺寸 */
  width: 100%;
  max-width: 400px; /* Medium */
  max-height: calc(100vh - 96px); /* 留出上下边距 */
  
  /* 圆角 - 28px */
  border-radius: 28px;
  
  /* 背景色 */
  background-color: var(--md-sys-color-surface);
  
  /* 阴影 */
  box-shadow: 
    0 11px 15px -7px rgba(0,0,0,0.2),
    0 9px 46px 8px rgba(0,0,0,0.14),
    0 18px 71px 10px rgba(0,0,0,0.12);
  
  /* 内边距 */
  padding: 24px;
  
  /* 布局 */
  display: flex;
  flex-direction: column;
  
  /* 位置 */
  position: relative;
  z-index: 1;
  
  /* 动画 */
  transform: scale(1);
  transition: transform 200ms ease;
}
```

## 🔴P1 Dialog 内部结构

```
┌────────────────────────────────────────┐
│                                        │
│  ┌──────────────────────────────────┐  │
│  │  [可选] Icon/Image               │  │
│  └──────────────────────────────────┘  │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │  Title (可选)                    │  │
│  └──────────────────────────────────┘  │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │  Content                         │  │
│  │  (支持滚动区域)                   │  │
│  └──────────────────────────────────┘  │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │  Actions (底部按钮)               │  │
│  └──────────────────────────────────┘  │
│                                        │
└────────────────────────────────────────┘
```

### CSS 布局

```css
.md3-dialog__surface {
  padding: 24px;
  display: flex;
  flex-direction: column;
  gap: 16px;
}

/* 标题区域 */
.md3-dialog__title {
  font-family: var(--md-sys-typescale-headline-small-font);
  font-size: 24px;
  font-weight: 400;
  color: var(--md-sys-color-on-surface);
  margin: 0;
  padding-right: 24px; /* 为关闭按钮留空间 */
  /* 如果有 icon */
  display: flex;
  align-items: center;
  gap: 16px;
}

/* 内容区域 */
.md3-dialog__content {
  flex: 1;
  overflow-y: auto;
  font-family: var(--md-sys-typescale-body-medium-font);
  font-size: 14px;
  line-height: 1.5;
  color: var(--md-sys-color-on-surface-variant);
  /* 内容内边距 */
  padding: 16px 0;
}

/* 操作区域 */
.md3-dialog__actions {
  display: flex;
  gap: 8px;
  justify-content: flex-end;
  flex-wrap: wrap;
  padding-top: 8px;
}

/* 操作按钮（使用 24px 圆角） */
.md3-dialog__actions .md3-button {
  border-radius: 24px;
}
```

## Dialog 类型

### Alert Dialog（警告对话框）

```css
/* 简单确认框 */
.md3-dialog--alert .md3-dialog__surface {
  max-width: 280px; /* Small */
  padding: 24px;
  text-align: center;
}

.md3-dialog--alert .md3-dialog__actions {
  justify-content: center;
}
```

### Simple Dialog（简单对话框）

```css
.md3-dialog--simple .md3-dialog__surface {
  max-width: 400px;
}

/* 带列表的对话框 */
.md3-dialog__content--list {
  padding: 0;
  max-height: 300px;
  overflow-y: auto;
}

.md3-dialog__list-item {
  display: flex;
  align-items: center;
  gap: 16px;
  padding: 12px 24px;
  cursor: pointer;
  transition: background-color 150ms ease;
}

.md3-dialog__list-item:hover {
  background-color: var(--md-sys-color-surface-container-high);
}
```

### Confirmation Dialog（确认对话框）

```css
/* 带单选按钮的确认对话框 */
.md3-dialog--confirmation .md3-dialog__surface {
  max-width: 560px; /* Large */
}
```

.md3-dialog--open .md3-dialog__scrim {

.md3-dialog--modal .md3-dialog__scrim {
  backdrop-filter: blur(4px);
  -webkit-backdrop-filter: blur(4px);

## Dialog 状态

```css
/* 关闭状态 */
.md3-dialog {
  opacity: 0;
  visibility: hidden;
  pointer-events: none;
}

/* 打开状态 */
.md3-dialog--open {
  opacity: 1;
  visibility: visible;
  pointer-events: auto;
}

/* 打开动画 */
.md3-dialog--open .md3-dialog__surface {
  transform: scale(1);
  animation: dialog-enter 200ms ease forwards;
}

@keyframes dialog-enter {
  from {
    transform: scale(0.95);
    opacity: 0;
  }
  to {
    transform: scale(1);
    opacity: 1;
  }
}

/* 关闭动画 */
.md3-dialog--closing .md3-dialog__surface {
  transform: scale(0.95);
  opacity: 0;
  transition: transform 150ms ease, opacity 150ms ease;
}
```

## 移动端规范

### 断点

```
移动端宽度：< 600px
```

### 移动端样式调整

```css
@media (max-width: 599px) {
  /* 移动端 Dialog 底部弹出 */
  .md3-dialog {
    align-items: flex-end;
    justify-content: center;
  }
  
  .md3-dialog__surface {
    max-width: 100%;
    max-height: calc(100vh * 0.9);
    border-radius: 28px 28px 0 0; /* 底部直角 */
    margin: 0;
    padding: 24px;
    padding-bottom: max(24px, env(safe-area-inset-bottom));
  }
  
  /* 底部弹出动画 */
  .md3-dialog--open .md3-dialog__surface {
    animation: dialog-slide-up 300ms ease forwards;
  }
  
  @keyframes dialog-slide-up {
    from {
      transform: translateY(100%);
    }
    to {
      transform: translateY(0);
    }
  }
}
```

### 移动端手势支持

```css
@media (max-width: 599px) {
  /* 下拉关闭 */
  .md3-dialog__surface--swipeable {
    touch-action: pan-y;
  }
  
  .md3-dialog--swiping-down .md3-dialog__surface {
    transform: translateY(var(--swipe-offset, 0));
    transition: none;
  }
}
```
