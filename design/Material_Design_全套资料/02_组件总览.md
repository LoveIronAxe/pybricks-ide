## 完整组件分类列表

### 1. Action（操作组件）
用户触发核心操作的组件

| 组件 | 功能说明 |
|------|---------|
| **Buttons** | 按钮：支持 5 种配色（elevated/filled/tonal/outlined/text）、5 种尺寸（XS/S/M/L/XL）、2 种形状（round/square）、默认 + toggle 变体 |
| **Button groups** | 按钮组：组织多个按钮，支持统一分组交互 |
| **Extended FABs** | 扩展浮动操作按钮：含图标 + 文字 + 可选副标题的大 FAB |
| **FAB menu** | FAB 菜单：从 FAB 展开多个相关操作 |
| **Floating action buttons (FABs)** | 浮动操作按钮：主操作入口 |
| **Icon buttons** | 图标按钮：单图标触发操作，IDE 工具栏最常用 |
| **Segmented buttons** | 分段按钮：一组选项切换视图/排序 |
| **Split buttons** | 分裂按钮：主按钮 + 展开菜单（如 Word 工具栏按钮） |

### 2. Date & Time（日期时间）
| 组件 | 功能说明 |
|------|---------|
| **Date pickers** | 日期选择器：选单个日期或日期范围 |
| **Time pickers** | 时间选择器：选具体时间 |

### 3. Loading & Progress（加载与进度）
| 组件 | 功能说明 |
|------|---------|
| **Loading indicators** | 加载指示器：短等待过程的旋转/脉冲动画 |
| **Progress indicators** | 进度条：显示实时进度（线性/圆形） |

### 4. Navigation（导航组件）
| 组件 | 功能说明 | IDE 典型场景 |
|------|---------|------------|
| **Navigation bar** | 底部导航栏：手机端视图切换 | 移动端模拟器 |
| **Navigation drawer** | 导航抽屉：侧滑/侧拉面板 | 文件资源管理器 |
| **Navigation rail** | 导航轨道：侧边图标+标签栏（桌面） | **IDE 活动栏**（类 VS Code） |

### 5. Sheets（面板）
| 组件 | 功能说明 |
|------|---------|
| **Bottom sheets** | 底部面板：从屏幕底部拉出的次要内容面板 |
| **Side sheets** | 侧面板：从屏幕边缘拉出的次要内容面板 |

### 6. Communication（通信组件）
| 组件 | 功能说明 | IDE 典型场景 |
|------|---------|------------|
| **Snackbar** | 轻量提示条：底部短暂消息 | "文件已保存"、"已复制到剪贴板" |
| **Tooltip** | 工具提示：悬停显示简短说明 | 工具栏图标悬停提示 |
| **Dialogs** | 对话框：模态弹窗，要求用户确认或输入 | 确认对话框、设置弹窗 |

### 7. Containment（容器组件）
| 组件 | 功能说明 |
|------|---------|
| **App bars** | 顶部应用栏：放置导航、标题、操作 |
| **Badges** | 徽章：显示通知数字或状态标记 |
| **Cards** | 卡片：承载单个主题的内容和操作 |
| **Carousel** | 轮播：可滚动的内容集合 |

### 8. Selection（选择组件）
| 组件 | 功能说明 | IDE 典型场景 |
|------|---------|------------|
| **Checkbox** | 复选框：多选 | 设置页多选项、文件多选 |
| **Chips** | 标签芯片：筛选/展示标签 | Git 标签筛选、文件类型筛选 |
| **Radio buttons** | 单选按钮：互斥选项 | 主题选择、视图模式切换 |
| **Sliders** | 滑块：范围内数值选择 | 字号调节、缩放比例 |
| **Switches** | 开关：开关 Toggle | 设置页开关项 |

### 9. Text Input（文本输入）
| 组件 | 功能说明 | IDE 典型场景 |
|------|---------|------------|
| **Text fields** | 文本输入框：单行/多行输入 | 命令面板、搜索框、代码输入 |
| **Search** | 搜索框：带搜索图标的专用输入 | 全局搜索、文件搜索 |

### 10. Layout（布局组件）
| 组件 | 功能说明 | IDE 典型场景 |
|------|---------|------------|
| **Lists** | 列表：连续的垂直文本/图像索引 | 文件树、Outline 视图、终端输出 |
| **Dividers** | 分割线：分组内容的细线 | 面板内分组分隔 |
| **Tabs** | 标签页：跨屏幕组织内容 | 多文件标签页 |
| **Toolbars** | 工具栏：显示当前页面常用操作 | 顶部菜单栏 |
| **Menus** | 菜单：临时显示选项列表 | 右键菜单、工具栏下拉菜单 |

## 组件变体速查（Buttons 为例）

MD3 组件的 5 种配色配置（以按钮为例）：
```
Elevated button    → 有阴影，略凸起（适合次要操作）
Filled button      → 实心填充，高对比度（适合主要操作）
Filled tonal       → 柔和填充，使用 Secondary 容器色
Outlined button    → 描边无填充，最轻量
Text button        → 无边框无填充，最轻量
```
