# 自定义配色方案

> 基于陛下指定的主色 #0071e3，为本项目 IDE 生成可直接复制的完整 Material Design 3 配色方案。

---

## 一、如何从单一主色生成完整调色板

### 1.1 三步生成法

**第一步：确定主色身份**
- 陛下指定色：`#0071e3`
- 经过分析：这是 **Blue 40**（MD3 标准主色位置）
- 色调：Blue (240°)，色相：87（最大色相）

**第二步：自动扩展色阶**
使用 MD3 的 13 级明度体系，从主色自动推导同色调的 13 个色阶：

| 色阶 | Hex 值 | 用途 |
|------|--------|------|
| 0 | `#000000` | 深色主题主色（极暗） |
| 10 | `#001a40` | 深色主题主容器 |
| 20 | `#002e6d` | 深色主题次要 |
| 30 | `#00468a` | 深色主题辅助容器 |
| **40** | **`#0071e3`** | **← 陛下指定的主色** |
| 50 | `#2584ff` | 次要主色 |
| 60 | `#4d97ff` | 悬停状态 |
| 70 | `#74abff` | 浅色辅助 |
| 80 | `#a8c8ff` | 深色主题主色（发光效果） |
| 90 | `#d3e4ff` | 浅色主题主容器 |
| 95 | `#ecf4ff` | 浅色表面背景 |
| 98 | `#f6faff` | 极浅背景 |
| 100 | `#ffffff` | 纯白 |

**第三步：生成 ON 颜色**
自动生成高对比度文字/图标颜色：
- ON-Primary (浅色主题)：`#ffffff`（白色）
- ON-Primary (深色主题)：`#003062`（深蓝）

### 1.2 辅助色的协调策略

辅助色与主色 Blue 协调的两种策略：

**策略 A：邻近色（推荐）**
- 主色 Blue (240°) → 辅助色选择 Cyan (180°) 或 Teal
- 视觉和谐，色调平滑过渡

**策略 B：同一色调降低饱和度**
- 辅助色仍使用 Blue 色调，但降低色相强度
- 适合简洁、统一感的设计

## 二、如何使用 Material Design Color Tool（官方工具）

### 2.1 工具访问

**访问地址**：https://material.io/resources/color

### 2.2 使用步骤

**Step 1：打开色彩工具**
- 访问 https://material.io/resources/color
- 点击 "Start"

**Step 2：输入主色**
- 在 Primary 颜色输入框输入 `#0071e3`
- 系统自动识别为 Blue 40
- 预览界面实时更新

**Step 3：调整辅助色（可选）**
- Secondary：点击选择，推荐使用 Teal 或 Cyan 色调
- Tertiary：点击选择，推荐使用 Purple 或 Pink 色调
- Error：保持 Red 色调（不可自定义色调）

**Step 4：生成调色板**
- 点击 "Export"
- 选择格式：CSS / Android XML / iOS Swift / Flutter
- 复制生成的代码

**Step 5：导出 Dark 主题**
- 切换右上角 Dark 模式开关
- 再次导出深色主题代码

### 2.3 替代工具：Material Theme Builder (Figma)

对于设计师，推荐使用 Figma 的 Material Theme Builder 插件：

1. 在 Figma 中安装 "Material Theme Builder" 插件
2. 创建一个画板，命名如 "Theme"
3. 在插件中输入 `#0071e3` 作为 Primary
4. 插件自动生成 Light/Dark 两套主题
5. 同步到 Material Design 组件库

## 五、主题切换实现方案

### 5.1 HTML 属性切换法（推荐）

```html
<!-- HTML -->
<html>
  <head>
    <link rel="stylesheet" href="md3-theme.css">
  </head>
  <body>
    <!-- 内容 -->
    <script>
      // 切换主题
      function toggleTheme() {
        const html = document.documentElement;
        const current = html.getAttribute('data-theme');
        html.setAttribute('data-theme', current === 'dark' ? 'light' : 'dark');
        localStorage.setItem('theme', html.getAttribute('data-theme'));
      }

      // 读取保存的主题
      const saved = localStorage.getItem('theme');
      if (saved) {
        document.documentElement.setAttribute('data-theme', saved);
      } else {
        // 跟随系统
        if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
          document.documentElement.setAttribute('data-theme', 'dark');
        }
      }
    </script>
  </body>
</html>
```

### 5.2 SCSS 混入方案（可选）

```scss
// _theme.scss
@mixin theme-light {
  --md-sys-color-primary: #0071e3;
  // ... 其他浅色变量
}

@mixin theme-dark {
  --md-sys-color-primary: #a8c8ff;
  // ... 其他深色变量
}

:root {
  @include theme-light();
}

[data-theme="dark"] {
  @include theme-dark();
}
```
