# Material Design 3 工具与设计资源

## 官方工具

### 1. Material Design Color Tool
**地址**：https://material.io/color

官方在线配色工具，可：
- 选择主题色调，自动生成 MD3 色彩系统的 5 组色调（Primary, Secondary, Tertiary, Neutral, Neutral-variant）
- 预览对比度是否满足 WCAG 标准
- 导出 CSS 变量 / Android / iOS / Flutter 代码

### 2. Material Theme Builder（Figma 官方插件）
**地址**：Figma 插件商店搜索 "Material Theme Builder"

由 Google 官方发布，可：
- 在 Figma 中直接生成 MD3 主题
- 支持自定义品牌色，自动映射到 MD3 语义色彩
- 导出 token（Design Token）供开发使用
- 支持 Web / Android / iOS / Flutter 多平台

### 3. Material Design 组件预览网站
**地址**：https://m3.material.io/components

每个组件页面底部有 **"Code"** 标签页，提供：
- HTML/CSS（Web 组件）
- Kotlin（Android）
- Swift（iOS）
- Dart（Flutter）

可直接复制参考实现。

### 4. Web 组件代码库
**地址**：https://github.com/material-components

Google 官方的 Web 组件库，含：
- `@material/web` - 官方 Web 组件（Nutton, Card, TextField 等）
- 每个组件都有详细 Demo 页面
- 支持 CDN 引入或 npm 安装

## Figma 资源

### 5. Figma MD3 Component Kit（官方）
**地址**：Figma 社区搜索 "Material Design 3"

Google 官方发布的 Figma 组件库，包含：
- 所有 MD3 组件（按钮、卡片、导航、对话框等）
- 完整的主题变量（色彩、字体、间距）
- 支持 light/dark 主题切换

### 6. Figma社区其他优质 MD3 资源
- "Material Design 3 Kit" by Priyanka
- "M3 Reference Figma" by Google（更完整的官方参考）

## 颜色工具

### 7. Leonardo（MD3 色彩系统在线工具）
**地址**：https://leonardocolor.io/

基于 Material Design 3 色彩系统构建的在线配色工具：
- 支持 MD3 色彩映射规则
- 实时预览配色效果
- 生成可访问的色板组合

## 图标资源

### 8. Material Design Icons
**CDN 地址**：
```
<!-- 完整图标字体 -->
<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">

<!-- Outlined 风格（MD3 推荐） -->
<link href="https://fonts.googleapis.com/icon?family=Material+Icons+Outlined" rel="stylesheet">

<!-- Rounded 风格 -->
<link href="https://fonts.googleapis.com/icon?family=Material+Icons+Round" rel="stylesheet">

<!-- Two Tone 风格 -->
<link href="https://fonts.googleapis.com/icon?family=Material+Icons+Two+Tone" rel="stylesheet">
```

**NPM 安装**：
```bash
npm install material-design-icons
```

**独立 SVG**：https://fonts.google.com/icons

包含 2000+ 图标，支持按风格（Filled, Outlined, Rounded, Two-tone）和关键字搜索。

### 9. Phosphor Icons（现代替代）
**地址**：https://phosphoricons.com/
轻量、支持多种风格（Regular, Bold, Fill），也常用于 MD3 项目。
