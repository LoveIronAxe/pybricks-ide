# 13_布局网格与间距系统.md

> Material Design 3 布局系统与间距规范，聚焦 Pybricks IDE 三栏布局的实际应用。

---

## 一、8dp 网格系统

### 1.1 核心规则

MD3 所有间距数值必须是 **8 的倍数**。这确保元素对齐、一致且节奏感强。

```
基准单位：8dp
允许值：4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48 / 64 …
```

**绝对禁止**：10dp、13dp、15dp、17dp、21dp 等非 8 倍数值。

### 1.2 基础间距对照表

| 名称 | 数值 | 适用场景 |
|------|------|---------|
| `xs` | **4dp** | 同一组件内部细粒度间距，如图标与文字的微调 |
| `sm` | **8dp** | 列表项之间、标签与内容的间距 |
| `md` | **16dp** | 组件内边距、组件间标准间距 |
| `lg` | **24dp** | 区块间间距、页面大屏边距 |
| `xl` | **32dp** | 大区块分隔、页面级段落间距 |
| `xxl` | **48dp** | 页面顶部/底部留白、巨大分隔 |

### 1.3 MD3 规范原文要点

> *"All measurements in Material Design 3 are multiples of 4dp and align to an 8dp grid."*
> 所有 MD3 尺寸均为 4dp 的倍数，且整体对齐 8dp 网格。

## 二、页面边距规范

### 2.1 断点定义（响应式窗口宽度）

| 窗口宽度 | 窗口级别 | 页面边距 |
|---------|---------|---------|
| **< 600dp** | Compact（紧凑） | **16dp** |
| **600dp ~ 840dp** | Medium（中等） | **24dp** |
| **> 840dp** | Expanded（展开） | **24dp** |

> 💡 本 Pybricks IDE 的实际断点：
> - `< 480dp`：手机，底部导航
> - `480dp ~ 768dp`：平板，缩小侧边栏
> - `> 768dp`：桌面，三栏完整布局

### 2.2 边距应用规则

```css
/* 大屏（Expanded / Medium）：边距 24dp */
.detail-panel {
  padding: 24px;
}

/* 小屏（Compact < 600dp）：边距 16dp */
@media (max-width: 600px) {
  .detail-panel {
    padding: 16px;
  }
}
```

## 三、组件间距具体用法

### 3.1 按间距值的推荐用法

| 间距 | 典型用法 |
|------|---------|
| **4dp** | 图标与相邻文字的微调对齐（±2dp 视觉修正） |
| **8dp** | 列表项间距（`margin-bottom: 4px` 或 `gap: 8px`）、Chip 内文字与边缘、导航项内部图标与标签间距 |
| **16dp** | 按钮内边距（高度 40dp 的 Filled Button 水平 padding 24px）、卡片内边距、组件间标准间距 |
| **24dp** | 页面大屏边距（24dp）、卡片与卡片之间的间距（`gap: 24px`）、列表面板内容与边缘 |
| **32dp** | Flexbox 主轴间隔（导航项间 gap）、区块标题与内容的间距 |
| **48dp** | 大区块间分隔（section 间隔）、表单组之间的间距 |

### 3.2 组件内部间距

```
卡片内边距（Card）：16dp
按钮内边距（Button）：水平 24dp，垂直 12dp（总高 40dp）
列表项内边距：12dp（水平）× 12dp（垂直）
输入框内边距：16dp
FAB 内边距：16dp（标准 56×56dp）
```

### 3.3 组件之间间距

```
紧密相关组件（如 Label + Input）：间距 16dp
中等关联组件（如 Button + Card）：间距 24dp
独立区块（如 Header + Content）：间距 32dp
```

## 四、Navigation Rail 宽度规范

### 4.1 三种宽度规格（MD3 规范）

| 类型 | 宽度 | 导航项尺寸 | 说明 |
|------|------|-----------|------|
| **Narrow（窄版）** | **56px** | 48×48dp 或 40×40dp | 仅图标，无标签，本项目采用 |
| **Medium（中等）** | **72px** | 56×56dp | 图标 + 短标签（仅一行）|
| **Standard（标准）** | **80px** | 56×56dp | 图标 + 单行标签 |
| **Extended（扩展）** | **256px** | 56px 高，宽度自适应 | 图标 + 多行标签 |

### 4.2 56px 窄版 Navigation Rail 的 MD3 规范依据

**规范来源**：MD3 Navigation Rail 组件规范

> *"A Navigation Rail can be 56dp wide when using only icons, without labels."*

**本项目侧边栏 56px 的构成**：
```
容器宽度：56px
导航项（item）：48×48dp（或 40×40dp）
图标：24×24dp
左右 padding：各 (56 - 48) / 2 = 4dp
```

```css
/* 本项目侧边栏 Navigation Rail */
.sidebar {
  width: 56px;
  min-width: 56px;
  display: flex;
  flex-direction: column;
  align-items: center;
  padding: 8px 0;
  gap: 4px;
}

/* 导航项 */
.sidebar-item {
  width: 48px;    /* 56px 容器内，图标 24px，两侧各留 4dp */
  height: 48px;
  border-radius: 12px;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* 选中指示条（左侧 3px 宽 pill）*/
.sidebar-item.active::before {
  content: '';
  position: absolute;
  left: 0;
  top: 50%;
  transform: translateY(-50%);
  width: 3px;
  height: 24px;
  background: var(--primary);  /* #0071e3 */
  border-radius: 0 3px 3px 0;
}
```

### 4.3 选中态指示器（Active Indicator）

MD3 Navigation Rail 的选中态为 **pill（胶囊）指示器**，不是左侧竖条：

```css
/* MD3 规范原版（pill 指示器）*/
.md3-navigation-rail__item--selected {
  background: var(--md-sys-color-secondary-container);
  color: var(--md-sys-color-on-secondary-container);
}

/* 指示器形状：16dp 圆角胶囊，覆盖整个 item */
.md3-navigation-rail__item {
  border-radius: 16px;
}
```

> ⚠️ 本项目用左侧 3px 竖条是定制风格，非 MD3 默认。规范默认是 item 整体变为主色容器色背景（pill 形状）。

## 五、三栏布局间距详解

### 5.1 整体布局结构

```
┌─────────┬──────────────────┬──────────────────────────────────┐
│  56px   │     260px        │         flex: 1                 │
│ Nav Rail│   列表面板        │         详情面板                 │
│         │  min:200         │                                 │
│  固定    │  max:360         │         自适应                   │
│         │                  │                                 │
│         │  边距: 12dp      │  padding: 24dp                  │
│         │  内间距: 12dp     │                                 │
└─────────┴──────────────────┴──────────────────────────────────┘
  容器间隔: 0（紧邻）  面板间隔: 1px（边框）  内容间隔: 24dp
```

### 5.2 三栏具体数值

| 区域 | 尺寸 | 间距规则 |
|------|------|---------|
| **Navigation Rail** | 固定 **56px** | 与列表面板紧邻（0px gap），border-right 分隔 |
| **列表面板（List Panel）** | 260px（min 200px / max 360px）| border-right 分隔；内部 padding: 12dp |
| **详情面板（Detail Panel）** | `flex: 1`（剩余全部）| 页面边距 padding: 24dp |
| **Nav Rail → 列表面板** | 0px gap + 1px border | 直接相邻，1px outline-variant 边框 |
| **列表面板 → 详情面板** | 无边框，自然过渡 | 无额外间距，详情 panel padding 决定起始位置 |

### 5.3 列表面板内部间距

```css
.list-panel {
  width: 260px;
  min-width: 200px;
  max-width: 360px;
  border-right: 1px solid var(--outline-variant);
}

.list-header {
  padding: 16px;        /* 面板内顶边距 */
  border-bottom: 1px solid var(--outline-variant);
}

.list-content {
  padding: 8px;         /* 面板内四周留白 */
  display: flex;
  flex-direction: column;
  gap: 4px;             /* 列表项之间 4dp 间距 */
}
```

### 5.4 详情面板内部间距

```css
.detail-panel {
  padding: 24px;        /* 页面级边距 24dp */
  display: flex;
  flex-direction: column;
  gap: 24px;            /* 主要区块间 24dp 间距 */
}
```

## 六、卡片间距与内边距

### 6.1 卡片尺寸规范

| 属性 | 数值 |
|------|------|
| 内边距（Padding） | **16dp** |
| 圆角（Border Radius） | **12dp** |
| 最小高度 | **88dp**（三行列表项卡片）|
| 与相邻卡片间距 | **16dp**（水平）或 **16dp**（垂直）|

### 6.2 卡片网格布局（关键断点）

**小屏（窗口宽度 < 600dp）：单列布局**
```
每行 1 张卡片
卡片宽度: 100% - 32dp（左右各 16dp 边距）
```

**大屏（窗口宽度 ≥ 600dp）：双列网格**
```
每行 2 张卡片
卡片宽度: (100% - 24dp - 16dp) / 2 = (100% - 40dp) / 2
（24dp = 面板左右各 12dp 间距，16dp = 卡片间 gap）
```

```css
/* 卡片容器网格 */
.card-grid {
  display: grid;
  gap: 16px;  /* 卡片间距 16dp */
  padding: 12px;
}

/* 小屏：单列 */
@media (max-width: 600px) {
  .card-grid {
    grid-template-columns: 1fr;
  }
}

/* 大屏：双列 */
@media (min-width: 600px) {
  .card-grid {
    grid-template-columns: repeat(2, 1fr);
  }
}
```

### 6.3 卡片内边距

```css
.card {
  padding: 16dp;           /* 内边距 MD3 规范 */
  border-radius: 12dp;     /* 圆角 MD3 规范 */
  background: var(--surface-variant);
  /* 无大阴影，MD3 依赖背景色区分层次 */
}
```

### 6.4 列表项卡片（List Item Card）间距

> 用于车辆列表、航点列表，与标准 Card 有别

```css
.list-item {
  padding: 12dp;            /* 内边距比标准卡片小（12dp vs 16dp）*/
  border-radius: 8dp;
  gap: 12dp;                /* 图标与文字间距 */
  margin-bottom: 4dp;       /* 列表项间距 4dp */
}
```

## 七、响应式断点详解

### 7.1 MD3 标准断点 vs 本项目断点

| 级别 | MD3 规范宽度 | 本项目实际断点 | 说明 |
|------|------------|--------------|------|
| **Compact** | **< 600dp** | **< 480dp** | 手机，底部导航，侧边栏移到顶部/底部 |
| **Medium** | **600dp ~ 840dp** | **480dp ~ 768dp** | 平板，侧边栏 48px，列表栏收窄 |
| **Expanded** | **> 840dp** | **> 768dp** | 桌面，三栏完整布局 |

### 7.2 各断点下布局变化

```
Compact (< 480dp)
├── Navigation Rail → 底部固定栏（60dp 高）
├── 列表面板 → 全宽（40% 高度）
└── 详情面板 → 全宽（剩余高度）

Medium (480dp ~ 768dp)
├── Navigation Rail → 左侧固定（48px 宽）
├── 列表面板 → 200px 宽（可折叠）
└── 详情面板 → flex: 1，padding: 16dp

Expanded (> 768dp)
├── Navigation Rail → 左侧固定（56px 宽）← 本项目标准
├── 列表面板 → 260px 宽（min 200 / max 360）
└── 详情面板 → flex: 1，padding: 24dp
```

### 7.3 卡片双列断点（🔴 P1 重点）

> **大屏一行两个卡片，小屏一列的断点：600dp**

```css
/* 🔴 卡片布局断点 */
/* < 600dp：小屏，单列 */
.card-grid {
  grid-template-columns: 1fr;
}

/* ≥ 600dp：大屏，双列 */
@media (min-width: 600px) {
  .card-grid {
    grid-template-columns: repeat(2, 1fr);
    gap: 16px;
  }
}

/* ≥ 1200dp（可选）：三列 */
@media (min-width: 1200px) {
  .card-grid {
    grid-template-columns: repeat(3, 1fr);
  }
}
```

> 📌 依据：MD3 响应式布局在 600dp 处从紧凑单列切换到中等双列网格。本项目将此规则应用于详情面板内的卡片网格。

### 7.4 响应式断点快速查询表

| 场景 | 断点宽度 | 布局变化 |
|------|---------|---------|
| 卡片双列 | ≥ 600dp | 单列 → 双列 |
| 侧边栏宽度 | < 768dp | 56px → 48px |
| 列表面板宽度 | < 768dp | 260px → 200px |
| 页面边距 | < 600dp | 24dp → 16dp |
| 侧边栏 → 底部栏 | < 480dp | 左侧侧栏 → 底部固定栏 |
| 详情面板 padding | < 768dp | 24dp → 16dp |

## 八、综合布局速查

### 8.1 本项目三栏布局速查

```
┌──────────────────────────────────────────────────────────┐
│ Navigation Rail          列表面板           详情面板      │
│ 56px 固定               260px (200~360)    flex: 1       │
│                                                           │
│ padding: 8px 0        border-right      padding: 24dp  │
│ gap: 4px               padding: 12px     gap: 24px       │
│                         gap: 4px                          │
└──────────────────────────────────────────────────────────┘
```

### 8.2 间距系统速查

| 间距 | 用途 |
|------|------|
| **4dp** | 列表项间距 4dp、Nav Rail 内 item gap |
| **8dp** | Nav Rail item 间距（gap: 4px + 自身 48px）、图标内部微调 |
| **12dp** | 列表项内边距、卡片网格内 padding |
| **16dp** | 卡片内边距、页面小屏边距、按钮内边距 |
| **24dp** | 页面大屏边距、组件间标准间距、卡片网格 gap |
| **32dp** | 大区块分隔间距 |
| **48dp** | 页面级大区块间隔 |
