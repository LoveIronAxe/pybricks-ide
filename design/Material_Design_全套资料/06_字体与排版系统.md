## 1. Type Scale 类型等级总览

Material Design 3（简称 M3）定义了 **5 大类别、13 个级别**的字体体系，遵循固定比例递增：

| 类别 | 级别 | 字号 | 行高 | 字重 | 字母间距 |
|------|------|------|------|------|----------|
| **Display** | Display Large | 57px | 64px | 400 | 0 |
| | Display Medium | 45px | 52px | 400 | 0 |
| | Display Small | 36px | 44px | 400 | 0 |
| **Headline** | Headline Large | 32px | 40px | 400 | 0 |
| | Headline Medium | 28px | 36px | 400 | 0 |
| | Headline Small | 24px | 32px | 400 | 0 |
| **Title** | Title Large | 22px | 28px | 400 | 0 |
| | Title Medium | 16px | 24px | 500 | 0.1px |
| | Title Small | 14px | 20px | 500 | 0.1px |
| **Body** | Body Large | 16px | 24px | 400 | 0.5px |
| | Body Medium | 14px | 20px | 400 | 0.25px |
| | Body Small | 12px | 16px | 400 | 0.4px |
| **Label** | Label Large | 14px | 20px | 500 | 0.1px |
| | Label Medium | 12px | 16px | 500 | 0.5px |
| | Label Small | 11px | 16px | 500 | 0.5px |

> 📐 **比例系统**：M3 使用 **Type Scale Ratio**（类型比例系数 ≈ 1.15），每升一级字号约放大 15%。

### 各类别设计意图

| 类别 | 设计用途 |
|------|----------|
| **Display** | 极少数使用，用于全屏欢迎页、品牌大字等极醒目场景 |
| **Headline** | 章节标题、弹窗标题、高亮文案 |
| **Title** | 卡片标题、列表项标题、导航项 |
| **Body** | 正文内容、说明文字、输入框文字 |
| **Label** | 按钮文字、标签、小图标下小字、辅助说明 |

## 2. 字体族

### Roboto（默认字体）

Roboto 是 Material Design 的默认字体家族，完整支持 Latin、Greek、Cyrillic 等字符集。

| 字重名称 | CSS font-weight | 适用场景 |
|----------|-----------------|----------|
| Thin | 100 | 极少使用 |
| Light | 300 | 极少使用 |
| Regular | 400 | 大部分正文 |
| Medium | 500 | 标题、按钮、标签 |
| Bold | 700 | 强调（慎用） |
| Black | 900 | 极少使用 |

### Roboto Flex（可变字体）

Roboto Flex 是 **Variable Font（可变字体）**，可通过 CSS 变量精细控制字体的粗细、宽度、倾角等轴。

**主要可变轴：**

| 轴名 | CSS 属性 | 范围 | 说明 |
|------|----------|------|------|
| wght | font-variation-settings | 100–900 | 字重（最常用） |
| wdth | font-stretch | 75%–125% | 字宽压缩/扩展 |
| slnt | font-style | -10–0 | 倾斜角度 |
| opsz | font-optical-sizing | 8–144 | 光学尺寸 |

```css
/* Roboto Flex 可变字体示例 */
.roboto-flex-demo {
  font-family: 'Roboto Flex', sans-serif;
  font-variation-settings: 'wght' 500, 'wdth' 100, 'slnt' 0, 'opsz' 24;
}
```

**推荐轴配置：**

```css
/* 标准正文 */
font-variation-settings: 'wght' 400, 'opsz' 14;
/* 中等强调 */
font-variation-settings: 'wght' 500, 'opsz' 16;
/* 标题 */
font-variation-settings: 'wght' 400, 'opsz' 32;
```

## 4. 字母间距（Letter-Spacing）规范

M3 中字母间距的使用规则：

| 级别 | letter-spacing | 适用场景 |
|------|----------------|----------|
| Display Large/Small | 0 | 不额外调整 |
| Headline | 0 | 不额外调整 |
| Title Medium/Small | +0.1px | 适度收紧 |
| Title Large | 0 | 不额外调整 |
| Body Large | +0.5px | 增强可读性 |
| Body Medium | +0.25px | 适中 |
| Body Small | +0.4px | 小字适当放宽 |
| Label Large | +0.1px | 按钮文字 |
| Label Medium | +0.5px | 辅助标签 |
| Label Small | +0.5px | 极小标签 |

### 字母间距使用原则

```css
/* ✅ 推荐做法：小字号适当增加字母间距 */
.label-small {
  letter-spacing: 0.5px; /* 11px 字号 + 0.5px 字间距 = 更清晰 */
}

/* ❌ 避免：大字号也加宽字间距，导致松散 */
.display-large {
  letter-spacing: 0; /* Display 级别不要加间距 */
}
```

## 5. 行高（Line-Height）规范

M3 行高设计遵循 **1.1–1.5 倍字号** 的比例：

| 级别 | 行高 | 倍率（相对于字号） |
|------|------|------------------|
| Display Large | 64px | ≈ 1.12× |
| Display Medium | 52px | ≈ 1.16× |
| Display Small | 44px | ≈ 1.22× |
| Headline Large | 40px | = 1.25× |
| Headline Medium | 36px | ≈ 1.29× |
| Headline Small | 32px | ≈ 1.33× |
| Title Large | 28px | ≈ 1.27× |
| Title Medium | 24px | = 1.5× |
| Title Small | 20px | ≈ 1.43× |
| Body Large | 24px | = 1.5× |
| Body Medium | 20px | ≈ 1.43× |
| Body Small | 16px | ≈ 1.33× |
| Label Large | 20px | ≈ 1.43× |
| Label Medium | 16px | ≈ 1.33× |
| Label Small | 16px | ≈ 1.45× |

### 行高设计原则

```css
/* ✅ 正文行高保持在 1.4–1.5 倍，适合阅读 */
.body-text {
  line-height: 1.5; /* 14px * 1.5 = 21px ≈ 20px 规范值 */
}

/* ✅ 标题行高可以稍紧，节省空间 */
.headline-text {
  line-height: 1.25; /* 标题不需要那么多行间距 */
}

/* ❌ 行高过低导致文字拥挤难读 */
.bad-example {
  line-height: 1.1; /* 太挤，阅读疲劳 */
}

/* ❌ 行高过高导致段落松散 */
.bad-example-2 {
  line-height: 2.0; /* 太稀，破坏视觉连贯性 */
}
```

## 6. IDE 应用建议

### 6.1 侧边栏图标下小字 → `Label` 级别

侧边栏项通常空间紧凑，字号偏小。

```css
/* 侧边栏图标下的标签文字 */
.sidebar-item-label {
  font-family: 'Roboto', sans-serif;
  font-size: 11px;
  line-height: 16px;
  font-weight: 500;
  letter-spacing: 0.5px;
  /* Material Design Label Small */
}
```

**推荐精确值：**

| 场景 | 推荐级别 | 字号 | 行高 | 字重 |
|------|----------|------|------|------|
| 侧边栏菜单项 | Label Medium | 12px | 16px | 500 |
| 侧边栏辅助说明 | Label Small | 11px | 16px | 500 |

### 6.2 列表标题 → `Title Large`

列表中的每行标题通常需要比正文更醒目。

```css
/* 列表标题 */
.list-item-title {
  font-family: 'Roboto', sans-serif;
  font-size: 22px;
  line-height: 28px;
  font-weight: 400;
  letter-spacing: 0;
  /* Material Design Title Large */
}
```

**列表内各类文字推荐：**

| 场景 | 推荐级别 | 字号 |
|------|----------|------|
| 主列表标题 | Title Large | 22px |
| 次级列表项标题 | Title Medium | 16px |
| 列表正文 | Body Medium | 14px |
| 列表辅助说明 | Label Medium | 12px |

### 6.3 弹窗标题 → `Headline Medium`

弹窗（Dialog）标题需要醒目但不夸张。

```css
/* 弹窗标题 */
.dialog-title {
  font-family: 'Roboto', sans-serif;
  font-size: 28px;
  line-height: 36px;
  font-weight: 400;
  letter-spacing: 0;
  /* Material Design Headline Medium */
}
```

**弹窗内各类文字推荐：**

| 场景 | 推荐级别 | 字号 |
|------|----------|------|
| 弹窗主标题 | Headline Medium | 28px |
| 弹窗副标题/描述 | Title Medium | 16px |
| 弹窗正文 | Body Medium | 14px |
| 弹窗按钮文字 | Label Large | 14px |

### 6.4 正文文本 → `Body Medium`

正文是 IDE 中使用最广泛的场景。

```css
/* IDE 正文字体 */
.ide-body-text {
  font-family: 'Roboto', 'Roboto Flex', sans-serif;
  font-size: 14px;
  line-height: 20px;
  font-weight: 400;
  letter-spacing: 0.25px;
  font-optical-sizing: auto;
  /* Material Design Body Medium */
}
```

## 7. 可直接复制的 CSS Type Scale

### 7.1 完整 CSS Type Scale（直接复制使用）

```css
/* ============================================================
   Material Design 3 Type Scale — 可直接复制使用
   字体：Roboto
   CDN: https://fonts.googleapis.com/css2?family=Roboto:wght@100;300;400;500;700;900&display=swap
   ============================================================ */

:root {
  /* Font Family */
  --md-sys-font-family: 'Roboto', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
  --md-sys-font-family-variable: 'Roboto Flex', 'Roboto', sans-serif;

  /* ============================================================
     Display
     ============================================================ */
  --md-sys-typescale-display-large-font:                 400 57px / 64px var(--md-sys-font-family);
  --md-sys-typescale-display-medium-font:                 400 45px / 52px var(--md-sys-font-family);
  --md-sys-typescale-display-small-font:                 400 36px / 44px var(--md-sys-font-family);

  /* ============================================================
     Headline
     ============================================================ */
  --md-sys-typescale-headline-large-font:                400 32px / 40px var(--md-sys-font-family);
  --md-sys-typescale-headline-medium-font:               400 28px / 36px var(--md-sys-font-family);
  --md-sys-typescale-headline-small-font:                400 24px / 32px var(--md-sys-font-family);

  /* ============================================================
     Title
     ============================================================ */
  --md-sys-typescale-title-large-font:                   400 22px / 28px var(--md-sys-font-family);
  --md-sys-typescale-title-medium-font:                  500 16px / 24px var(--md-sys-font-family);
  --md-sys-typescale-title-small-font:                   500 14px / 20px var(--md-sys-font-family);

  /* ============================================================
     Body
     ============================================================ */
  --md-sys-typescale-body-large-font:                    400 16px / 24px var(--md-sys-font-family);
  --md-sys-typescale-body-medium-font:                   400 14px / 20px var(--md-sys-font-family);
  --md-sys-typescale-body-small-font:                     400 12px / 16px var(--md-sys-font-family);

  /* ============================================================
     Label
     ============================================================ */
  --md-sys-typescale-label-large-font:                    500 14px / 20px var(--md-sys-font-family);
  --md-sys-typescale-label-medium-font:                   500 12px / 16px var(--md-sys-font-family);
  --md-sys-typescale-label-small-font:                    500 11px / 16px var(--md-sys-font-family);

  /* 字母间距修正（补充 letter-spacing） */
  --md-sys-typescale-body-large-letter-spacing:          0.5px;
  --md-sys-typescale-body-medium-letter-spacing:         0.25px;
  --md-sys-typescale-body-small-letter-spacing:           0.4px;
  --md-sys-typescale-title-medium-letter-spacing:        0.1px;
  --md-sys-typescale-title-small-letter-spacing:          0.1px;
  --md-sys-typescale-label-large-letter-spacing:         0.1px;
  --md-sys-typescale-label-medium-letter-spacing:        0.5px;
  --md-sys-typescale-label-small-letter-spacing:          0.5px;
}

/* ============================================================
   CSS Classes — 直接添加 class 使用
   ============================================================ */

/* Display */
.display-large {
  font: var(--md-sys-typescale-display-large-font);
}
.display-medium {
  font: var(--md-sys-typescale-display-medium-font);
}
.display-small {
  font: var(--md-sys-typescale-display-small-font);
}

/* Headline */
.headline-large {
  font: var(--md-sys-typescale-headline-large-font);
}
.headline-medium {
  font: var(--md-sys-typescale-headline-medium-font);
}
.headline-small {
  font: var(--md-sys-typescale-headline-small-font);
}

/* Title */
.title-large {
  font: var(--md-sys-typescale-title-large-font);
}
.title-medium {
  font: var(--md-sys-typescale-title-medium-font);
  letter-spacing: var(--md-sys-typescale-title-medium-letter-spacing);
}
.title-small {
  font: var(--md-sys-typescale-title-small-font);
  letter-spacing: var(--md-sys-typescale-title-small-letter-spacing);
}

/* Body */
.body-large {
  font: var(--md-sys-typescale-body-large-font);
  letter-spacing: var(--md-sys-typescale-body-large-letter-spacing);
}
.body-medium {
  font: var(--md-sys-typescale-body-medium-font);
  letter-spacing: var(--md-sys-typescale-body-medium-letter-spacing);
}
.body-small {
  font: var(--md-sys-typescale-body-small-font);
```

### 7.3 SCSS / CSS Variables 混合版本（推荐）

```scss
// ============================================================
// Material Design 3 Type Scale — SCSS Variables 版本
// ============================================================

$font-family-base: 'Roboto', -apple-system, BlinkMacSystemFont, sans-serif;
$font-family-variable: 'Roboto Flex', 'Roboto', sans-serif;

// M3 Type Scale
$type-scale: (
  'display-large':   (size: 57px, line-height: 64px, weight: 400, tracking: 0),
  'display-medium':   (size: 45px, line-height: 52px, weight: 400, tracking: 0),
  'display-small':    (size: 36px, line-height: 44px, weight: 400, tracking: 0),
  'headline-large':  (size: 32px, line-height: 40px, weight: 400, tracking: 0),
  'headline-medium': (size: 28px, line-height: 36px, weight: 400, tracking: 0),
  'headline-small':  (size: 24px, line-height: 32px, weight: 400, tracking: 0),
  'title-large':     (size: 22px, line-height: 28px, weight: 400, tracking: 0),
  'title-medium':    (size: 16px, line-height: 24px, weight: 500, tracking: 0.1px),
  'title-small':      (size: 14px, line-height: 20px, weight: 500, tracking: 0.1px),
  'body-large':      (size: 16px, line-height: 24px, weight: 400, tracking: 0.5px),
  'body-medium':     (size: 14px, line-height: 20px, weight: 400, tracking: 0.25px),
  'body-small':       (size: 12px, line-height: 16px, weight: 400, tracking: 0.4px),
  'label-large':      (size: 14px, line-height: 20px, weight: 500, tracking: 0.1px),
  'label-medium':     (size: 12px, line-height: 16px, weight: 500, tracking: 0.5px),
  'label-small':      (size: 11px, line-height: 16px, weight: 500, tracking: 0.5px),
);

// Mixin for generating type scale classes
@mixin m3-type($name) {
  $spec: map-get($type-scale, $name);
  @if $spec {
    font-family: $font-family-base;
    font-size: map-get($spec, 'size');
    line-height: map-get($spec, 'line-height');
    font-weight: map-get($spec, 'weight');
    letter-spacing: map-get($spec, 'tracking');
  }
}

// Usage:
// .headline-medium { @include m3-type('headline-medium'); }
// .body-text       { @include m3-type('body-medium'); }
// .sidebar-label   { @include m3-type('label-small'); }
```
