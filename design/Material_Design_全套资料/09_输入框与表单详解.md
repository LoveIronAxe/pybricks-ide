# 09_输入框与表单详解

输入框是 IDE 中接收用户输入的核心组件，包括文本字段、复选框、开关和滑块。

---

## 🔴P1 Text Field（文本字段）

MD3 有两种文本字段变体：Outlined（推荐）和 Filled。

---

## 🔴P1 Outlined Text Field（轮廓文本字段）

**MD3 默认推荐的文本字段样式。**

### 尺寸规范

```
高度：56px
圆角：4px
标签内边距：16px（水平）
边框宽度：1px
```

### CSS 实现

```css
.md3-text-field--outlined {
  position: relative;
  height: 56px;
  border-radius: 4px;
}

/* 外层容器（用于边框） */
.md3-text-field--outlined .md3-text-field__border {
  position: absolute;
  inset: 0;
  border: 1px solid var(--md-sys-color-outline);
  border-radius: 4px;
  pointer-events: none;
  transition: border-color 200ms ease;
}

/* 输入框 */
.md3-text-field--outlined input,
.md3-text-field--outlined textarea {
  width: 100%;
  height: 100%;
  padding: 16px;
  padding-top: 24px; /* 为标签留出空间 */
  border: none;
  background: transparent;
  font-family: var(--md-sys-typescale-body-large-font);
  font-size: 16px;
  color: var(--md-sys-color-on-surface);
  outline: none;
}

/* 浮动标签 */
.md3-text-field--outlined label {
  position: absolute;
  left: 16px;
  top: 50%;
  transform: translateY(-50%);
  font-family: var(--md-sys-typescale-body-large-font);
  font-size: 16px;
  color: var(--md-sys-color-on-surface-variant);
  pointer-events: none;
  transition: all 150ms ease;
}

/* 聚焦时的浮动标签 */
.md3-text-field--outlined input:focus ~ label,
.md3-text-field--outlined input:not(:placeholder-shown) ~ label,
.md3-text-field--outlined textarea:focus ~ label,
.md3-text-field--outlined textarea:not(:placeholder-shown) ~ label {
  top: 8px;
  transform: translateY(0);
  font-size: 12px;
}
```

### 状态样式

```css
/* 默认状态 */
.md3-text-field--outlined .md3-text-field__border {
  border-color: var(--md-sys-color-outline);
}

/* 悬停状态 */
.md3-text-field--outlined:hover .md3-text-field__border {
  border-color: var(--md-sys-color-on-surface);
}

/* 聚焦状态 */
.md3-text-field--outlined:focus-within .md3-text-field__border {
  border-color: var(--md-sys-color-primary);
  border-width: 2px;
}

/* 禁用状态 */
.md3-text-field--outlined:disabled {
  opacity: 0.38;
  pointer-events: none;
}

.md3-text-field--outlined:disabled .md3-text-field__border {
  border-color: var(--md-sys-color-on-surface);
  border-style: dashed;
}

/* 错误状态 */
.md3-text-field--outlined.error .md3-text-field__border {
  border-color: var(--md-sys-color-error);
}

.md3-text-field--outlined.error label {
  color: var(--md-sys-color-error);
}
```

## Filled Text Field（填充文本字段）

**使用底部边框 + 填充背景。**

### CSS 实现

```css
.md3-text-field--filled {
  position: relative;
  height: 56px;
  border-radius: 4px 4px 0 0;
  background-color: var(--md-sys-color-surface-container-high);
}

.md3-text-field--filled input {
  width: 100%;
  height: 100%;
  padding: 24px 16px 8px 16px;
  border: none;
  background: transparent;
  font-size: 16px;
  color: var(--md-sys-color-on-surface);
  outline: none;
}

/* 底部边框 */
.md3-text-field--filled::after {
  content: '';
  position: absolute;
  bottom: 0;
  left: 0;
  right: 0;
  height: 1px;
  background-color: var(--md-sys-color-outline);
  transition: background-color 200ms ease;
}

.md3-text-field--filled:focus-within::after {
  height: 2px;
  background-color: var(--md-sys-color-primary);
}

/* 悬停状态 */
.md3-text-field--filled:hover {
  background-color: var(--md-sys-color-surface-container-highest);
}
```

## 🔴P1 Checkbox（复选框）

### 尺寸规范

```
尺寸：18px × 18px（可选 20px × 20px）
圆角：2px（内部 icon 的圆角）
边框宽度：2px
间距：8px（与标签）
```

### CSS 实现

```css
.md3-checkbox {
  display: inline-flex;
  align-items: center;
  gap: 8px;
  cursor: pointer;
  user-select: none;
}

/* 隐藏原生 checkbox */
.md3-checkbox input {
  position: absolute;
  width: 0;
  height: 0;
  opacity: 0;
}

/* 复选框容器 */
.md3-checkbox .md3-checkbox__box {
  width: 18px;
  height: 18px;
  border: 2px solid var(--md-sys-color-outline);
  border-radius: 2px;
  display: flex;
  align-items: center;
  justify-content: center;
  transition: all 150ms ease;
  flex-shrink: 0;
}

/* 选中时的图标 */
.md3-checkbox .md3-checkbox__icon {
  opacity: 0;
  transform: scale(0);
  transition: all 150ms ease;
  color: var(--md-sys-color-on-secondary-container);
}

/* 选中状态 */
.md3-checkbox input:checked + .md3-checkbox__box {
  background-color: var(--md-sys-color-secondary-container);
  border-color: var(--md-sys-color-secondary-container);
}

.md3-checkbox input:checked + .md3-checkbox__box .md3-checkbox__icon {
  opacity: 1;
  transform: scale(1);
}

/* 悬停状态 */
.md3-checkbox:hover .md3-checkbox__box {
  border-color: var(--md-sys-color.on-surface);
}

/* 聚焦状态 */
.md3-checkbox input:focus-visible + .md3-checkbox__box {
  outline: 2px solid var(--md-sys-color-focus-ring);
  outline-offset: 2px;
}

/* 禁用状态 */
.md3-checkbox input:disabled + .md3-checkbox__box {
  border-color: var(--md-sys-color-on-surface);
  opacity: 0.38;
}

.md3-checkbox input:disabled:checked + .md3-checkbox__box {
  background-color: var(--md-sys-color-on-surface);
  opacity: 0.38;
}
```

## 🔴P1 Switch（开关）

### 尺寸规范

```
轨道宽度：52px
轨道高度：32px
thumb 直径：24px
thumb 位移：8px（关闭）/ 20px（开启）
圆角：16px（轨道）
```

### CSS 实现

```css
.md3-switch {
  display: inline-flex;
  align-items: center;
  cursor: pointer;
  user-select: none;
}

.md3-switch input {
  position: absolute;
  width: 0;
  height: 0;
  opacity: 0;
}

/* 轨道 */
.md3-switch .md3-switch__track {
  width: 52px;
  height: 32px;
  border-radius: 16px;
  background-color: var(--md-sys-color-surface-variant);
  border: 2px solid var(--md-sys-color-outline);
  position: relative;
  transition: all 200ms ease;
  flex-shrink: 0;
}

/* Thumb */
.md3-switch .md3-switch__thumb {
  position: absolute;
  top: 2px;
  left: 2px;
  width: 24px;
  height: 24px;
  border-radius: 50%;
  background-color: var(--md-sys-color-outline);
  transition: all 200ms ease;
}

/* 选中状态 */
.md3-switch input:checked + .md3-switch__track {
  background-color: var(--md-sys-color-secondary-container);
  border-color: var(--md-sys-color-secondary-container);
}

.md3-switch input:checked + .md3-switch__track .md3-switch__thumb {
  left: 24px; /* 52 - 24 - 4(边框+间距) */
  background-color: var(--md-sys-color-on-secondary-container);
}

/* 悬停状态 */
.md3-switch:hover .md3-switch__track {
  border-color: var(--md-sys-color-on-surface);
}

/* 聚焦状态 */
.md3-switch input:focus-visible + .md3-switch__track {
  outline: 2px solid var(--md-sys-color-focus-ring);
  outline-offset: 2px;
}

/* 禁用状态 */
.md3-switch input:disabled + .md3-switch__track {
  opacity: 0.38;
  pointer-events: none;
}
```

## Slider（滑块）

### 尺寸规范

```
轨道高度：4px（静止）/ 6px（交互）
Thumb 直径：20px
Thumb 触摸区域：48px × 48px
圆角：10px（Thumb）
```

### CSS 实现

```css
.md3-slider {
  display: flex;
  align-items: center;
  height: 48px; /* 触摸区域高度 */
  cursor: pointer;
}

.md3-slider input {
  position: absolute;
  width: 0;
  height: 0;
  opacity: 0;
}

/* 轨道背景 */
.md3-slider .md3-slider__track {
  flex: 1;
  height: 4px;
  border-radius: 2px;
  background-color: var(--md-sys-color-surface-variant);
  position: relative;
}

/* 激活轨道 */
.md3-slider .md3-slider__track-fill {
  position: absolute;
  left: 0;
  top: 0;
  height: 100%;
  border-radius: 2px;
  background-color: var(--md-sys-color-primary);
  transition: width 100ms ease;
}

/* Thumb */
.md3-slider .md3-slider__thumb {
  position: absolute;
  top: 50%;
  transform: translate(-50%, -50%);
  width: 20px;
  height: 20px;
  border-radius: 50%;
  background-color: var(--md-sys-color-primary);
  transition: left 100ms ease, transform 100ms ease;
  /* 触摸区域 */
  width: 48px;
  height: 48px;
  top: 50%;
  margin-top: -24px;
  background: transparent;
}

.md3-slider .md3-slider__thumb::after {
  content: '';
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  width: 20px;
  height: 20px;
  border-radius: 50%;
  background-color: var(--md-sys-color-primary);
}

/* Hover */
.md3-slider:hover .md3-slider__track {
  height: 6px;
}

/* 聚焦 */
.md3-slider input:focus-visible ~ .md3-slider__thumb::after {
  outline: 2px solid var(--md-sys-color-focus-ring);
  outline-offset: 2px;
}

/* 禁用 */
.md3-slider input:disabled {
  pointer-events: none;
}

.md3-slider input:disabled ~ .md3-slider__track {
  opacity: 0.38;
}

.md3-slider input:disabled ~ .md3-slider__thumb::after {
  background-color: var(--md-sys-color-on-surface);
}
```

## 表单布局间距

```css
/* 标签与输入框间距 */
.md3-form-field {
  display: flex;
  flex-direction: column;
  gap: 4px;
}

/* 表单项间距 */
.md3-form-field + .md3-form-field {
  margin-top: 16px;
}

/* 标签样式 */
.md3-form-field__label {
  font-family: var(--md-sys-typescale-body-small-font);
  font-size: 12px;
  color: var(--md-sys-color-on-surface-variant);
}

/* 辅助文本 */
.md3-form-field__helper {
  font-family: var(--md-sys-typescale-body-small-font);
  font-size: 12px;
  color: var(--md-sys-color-on-surface-variant);
  margin-top: 4px;
}

/* 错误文本 */
.md3-form-field__error {
  font-family: var(--md-sys-typescale-body-small-font);
  font-size: 12px;
  color: var(--md-sys-color-error);
  margin-top: 4px;
}
```
