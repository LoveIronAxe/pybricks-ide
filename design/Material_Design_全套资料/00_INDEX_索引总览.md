## 🔴 P1 本项目重点文件（必读）

### 1. 20_IDE项目应用摘要.md ⭐⭐⭐
**第一优先 — 本项目精华汇总**，专为 Pybricks IDE 提炼，开箱即用。

- ✅ **完整 CSS 变量集**：本项目暗色主题一键复制（`:root` 变量块）
- ✅ **侧边栏样式**：56px 窄版 Navigation Rail，含选中态指示条
- ✅ **按钮代码**：Filled Button / Outlined Button 完整样式（含 hover/active）
- ✅ **输入框/开关/弹窗/FAB/Snackbar**：各组件精选代码

- ✅ **每个 IDE 页面的组件清单**：车辆配置页 / 地图编辑页 / 设置页逐一列出
- ✅ **不建议使用的内容清单**：避免引入过度设计的明确指引

### 2. 06_字体与排版系统.md ⭐⭐⭐
**第二优先 — 字体系统权威参考**（611 行，内容最详尽）

- ✅ **完整 Type Scale 表格**：Display / Headline / Title / Body / Label 全等级字号/字重/行高

### 3. 07_按钮组件详解.md ⭐⭐⭐
**第三优先 — 按钮专题圣经**（277 行，覆盖所有按钮类型）

### 4. 09_输入框与表单详解.md ⭐⭐⭐

## 开发流程建议

```
第 1 步：通读 20_IDE项目应用摘要.md（30 分钟）
  → 了解本项目需要哪些 MD3 组件，建立整体认知

第 2 步：确定需要的组件类型
  → 按钮 → 07 / 输入框 → 09 / 弹窗 → 10 / 动效 → 12

第 3 步：具体开发时从对应文件拷贝代码
  → 所有文件都包含可直接运行的 CSS/HTML/JS 代码

第 4 步：开发完成后过一遍 19_MD3最佳实践与常见错误.md
  → 检查是否有违背 MD3 规范的地方
```
