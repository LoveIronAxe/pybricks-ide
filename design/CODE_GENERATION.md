# Pybricks 机器车 IDE — 代码生成系统设计

> 本文档说明如何将用户在网页 UI 上的配置（vehicles.json）转换为可运行的 Python 代码。

---

## 一、JSON → Python 代码映射

### 1.1 vehicles.json 结构总览

```json
{
  "vehicles": [
    {
      "id": "uuid",
      "name": "车辆名称",
      "hub": {
        "type": "PrimeHub",
        "ports": {
          "A": { "device": "Motor", "config": { "direction": "CCW", "initAngle": 0 } },
          "B": { "device": "Motor", "config": { "direction": "CW",  "initAngle": 0 } },
          "E": { "device": "ColorSensor", "config": {} }
        }
      },
      "wheelGroups": [
        {
          "id": "uuid",
          "leftMotor":  { "port": "A", "initAngle": 0, "reversed": true  },
          "rightMotor": { "port": "B", "initAngle": 0, "reversed": false },
          "wheelDiameter": 62,
          "wheelWidth":   20
        }
      ],
      "lineFollowConfigs": [
        {
          "id": "uuid",
          "sensorPort": "E",
          "lineColor":      "#000000",
          "backgroundColor": "#FFFFFF",
          "sensorSide": "left",
          "threshold":    35,
          "kp":           1.5,
          "scanIntervalMs": 10
        }
      ]
    }
  ]
}
```

### 1.2 字段映射表（JSON → Python）

| vehicles.json 路径 | 示例值 | 生成的 Python 代码片段 |
|---|---|---|
| `hub.type` | `"PrimeHub"` | `from pybricks.hubs import PrimeHub` |
| `hub.ports.A.device` | `"Motor"` | `from pybricks.pupdevices import Motor` |
| `hub.ports.E.device` | `"ColorSensor"` | `from pybricks.pupdevices import ColorSensor` |
| `hub.ports[port].config.direction` | `"CCW"` | `Direction.COUNTERCLOCKWISE` |
| `hub.ports[port].config.direction` | `"CW"` | `Direction.CLOCKWISE` |
| `wheelGroups[].leftMotor.port` | `"A"` | `LEFT_MOTOR_PORT = Port.A` |
| `wheelGroups[].rightMotor.port` | `"B"` | `RIGHT_MOTOR_PORT = Port.B` |
| `wheelGroups[].wheelDiameter` | `62` | `wheel_diameter=62`（传给 DriveBase） |
| `wheelGroups[].leftMotor.reversed` | `true` | `Direction.COUNTERCLOCKWISE` |
| `wheelGroups[].leftMotor.reversed` | `false` | `Direction.CLOCKWISE`（右侧默认） |
| `lineFollowConfigs[].sensorPort` | `"E"` | `LINE_SENSOR_PORT = Port.E` |
| `lineFollowConfigs[].threshold` | `35` | 生成 `calibrate()` 中的阈值常量 |
| `lineFollowConfigs[].kp` | `1.5` | `kp * error` 中的比例系数 |
| `lineFollowConfigs[].scanIntervalMs` | `10` | `wait(wait_ms)` 中的采样周期 |

### 1.3 设备类型 → Import 映射

| UI 设备名 | vehicles.json 值 | 生成的 Import 语句 |
|---|---|---|
| 中/大型电机 | `"Motor"` | `from pybricks.pupdevices import Motor` |
| 颜色传感器 | `"ColorSensor"` | `from pybricks.pupdevices import ColorSensor` |
| 超声波传感器 | `"UltrasonicSensor"` | `from pybricks.pupdevices import UltrasonicSensor` |
| 力传感器 | `"ForceSensor"` | `from pybricks.pupdevices import ForceSensor` |
| Prime Hub | `"PrimeHub"` | `from pybricks.hubs import PrimeHub` |

### 1.4 方向映射

| UI 选择 | JSON 值 | 生成的 Python 常量 |
|---|---|---|
| 顺时针 | `"CW"` | `Direction.CLOCKWISE` |
| 逆时针 | `"CCW"` | `Direction.COUNTERCLOCKWISE` |

## 二、模板引擎方案

### 2.1 方案对比

| 方案 | 描述 | 优点 | 缺点 |
|---|---|---|---|
| **字符串拼接** | `"LEFT_MOTOR_PORT = Port." + port` | 简单直接，无依赖 | 容易出错，难以维护，换行/缩进需手动处理 |
| **Python string.Template** | `$VAR` 占位符 + `Template.substitute()` | 标准库无需安装，语法简单 | 不支持条件、循环，数据结构支持弱 |
| **jinja2** | `{{ var }}` + `{% for %}` 块 | 功能强大，支持条件/循环/宏，生态成熟 | 需要安装（`pip install jinja2`），模板语法稍重 |
| **AST 操作** | `ast.NodeTransformer` 修改 Python AST | 精确到语法树，可做静态分析 | 复杂度高，对字符串模板区域不友好 |

### 2.2 推荐方案：jinja2

**原因**：
1. 条件块 `{% if motor %}` 可以处理"未配置电机"跳过相关代码
2. 循环 `{% for device in devices %}` 避免重复代码
3. 宏 `{% macro init_motor(port) %}` 提取复用片段
4. 在 Web 端（Node.js/Python 后端）都有成熟实现

**安装**：
```bash
# Python 后端
pip install jinja2

# Node.js 前端（可选，前端只负责 JSON，后端生成）
npm install jinja2
```

### 2.3 模板文件结构

```
templates/
├── base.py.jinja2        # 基模板（文件头、import、init 框架）
├── motor.py.jinja2       # 电机相关代码片段
├── sensor.py.jinja2      # 传感器相关代码片段
├── line_follow.py.jinja2 # 巡线逻辑片段
└── macros.py.jinja2      # 共享宏（端口常量、设备实例化等）
```

### 2.4 模板示例

#### 主模板（base.py.jinja2）

```jinja2
# ============================================================
# Auto-generated by Pybricks IDE — DO NOT EDIT MANUALLY
# Vehicle: {{ vehicle.name }}
# Generated at: {{ timestamp }}
# ============================================================

{# === IMPORTS === #}
{# --- Hub --- #}
{% if vehicle.hub.type == "PrimeHub" %}
from pybricks.hubs import PrimeHub
{% endif %}

{# --- Devices --- #}
{% if has_motor %}
from pybricks.pupdevices import Motor
{% endif %}
{% if has_color_sensor %}
from pybricks.pupdevices import ColorSensor
{% endif %}
{% if has_ultrasonic %}
from pybricks.pupdevices import UltrasonicSensor
{% endif %}

{# --- Parameters & Robotics --- #}
from pybricks.parameters import Port, Direction, Stop
{% if has_drivebase %}
from pybricks.robotics import DriveBase
{% endif %}
from pybricks.tools import wait

{# === PORT CONSTANTS === #}
{% for port, mapping in port_mappings.items() %}
{{ mapping.constant_name }} = Port.{{ port }}
{% endfor %}

MAX_MOTOR_SPEED = 1180


{# === GLOBAL VARIABLES === #}
hub          = None
{% for motor in motors %}
{{ motor.var_name }} = None
{% endfor %}
{% if has_drivebase %}
drive_base   = None
{% endif %}


{# === INIT FUNCTION === #}
def init():
    global hub{% for motor in motors %}, {{ motor.var_name }}{% endfor %}{% if has_drivebase %}, drive_base{% endif %}

    hub = PrimeHub()

    {% for motor in motors %}
    {{ motor.var_name }} = Motor({{ motor.port_const }}, Direction.{{ motor.direction }})
    {% endfor %}

    {% if has_color_sensor %}
    line_sensor = ColorSensor(LINE_SENSOR_PORT)
    {% endif %}

    {% if has_drivebase %}
    drive_base = DriveBase(
        {{ left_motor.var_name }},
        {{ right_motor.var_name }},
        wheel_diameter={{ wheel_diameter }},
        axle_track={{ axle_track }}
    )
    {% endif %}

    while not hub.imu.ready():
        wait(10)

    {% if has_drivebase %}
    # Calibrate threshold for line following
    global threshold
    threshold = calibrate()
    {% endif %}


{# === CALIBRATE FUNCTION === #}
{% if has_color_sensor %}
def calibrate():
    white_sum = 0
    for _ in range(10):
        white_sum += line_sensor.reflection()
        wait(20)
    white_ref = white_sum // 10
    wait(500)
    black_sum = 0
    for _ in range(10):
        black_sum += line_sensor.reflection()
        wait(20)
    black_ref = black_sum // 10
    return (white_ref + black_ref) // 2
{% endif %}


{# === FOLLOW LINE FUNCTION === #}
{% if has_line_follow %}
def follow_line(distance, enable_lf, base_speed, kp={{ default_kp }}, wait_ms={{ default_wait_ms }}, stop_on_end=True):
    base = base_speed
    max_s = int(MAX_MOTOR_SPEED * 0.9)
    if base > max_s:
        base = max_s
    min_s = int(MAX_MOTOR_SPEED * -0.3)
    drive_base.reset()
    while abs(drive_base.distance()) < distance:
        if enable_lf:
            reflection = line_sensor.reflection()
            error = reflection - threshold
            turn_rate = kp * error
            left_speed  = base - turn_rate
            right_speed = base + turn_rate
            left_speed  = max(min_s, min(max_s, left_speed))
            right_speed = max(min_s, min(max_s, right_speed))
            {{ left_motor.var_name }}.run(left_speed)
            {{ right_motor.var_name }}.run(right_speed)
        else:
            drive_base.drive(base, 0)
        wait(wait_ms)
    if stop_on_end:
        {{ left_motor.var_name }}.stop()
        {{ right_motor.var_name }}.stop()
{% endif %}


{# === TURN FUNCTION === #}
{% if has_drivebase %}
def turn(angle, radius, speed, stop_on_end=True):
    max_s = int(MAX_MOTOR_SPEED * 0.9)
    if speed > max_s:
        speed = max_s
    if radius == 0:
        drive_base.turn(angle, then=Stop.HOLD if stop_on_end else Stop.COAST)
    else:
        drive_base.curve(radius, angle, then=Stop.HOLD if stop_on_end else Stop.COAST)
    if stop_on_end:
        wait(100)
{% endif %}


{# === USER CODE AREA — PRESERVED ACROSS GENERATIONS === #}
# === USER CODE START ===
{{ user_code_initial }}
# === USER CODE END ===

{# === ENTRY POINT === #}
init()
```

## 三、代码生成器架构

### 3.1 整体流程

```
┌─────────────────────┐
│   用户在网页 UI      │
│   配置车辆参数        │
│   （vehicles.json）  │
└──────────┬──────────┘
           │ 保存到 localStorage
           ▼
┌─────────────────────┐
│   vehicles.json     │
│   (完整配置数据)      │
└──────────┬──────────┘
           │ 传递给代码生成器
           ▼
┌──────────────────────────────────────┐
│         代码生成器（CodeGenerator）    │
│                                      │
│  1. 解析 vehicles.json                │
│  2. 收集 Port 常量                    │
│  3. 识别设备类型 → 决定 import         │
│  4. 提取 wheelGroup → DriveBase 参数  │
│  5. 提取 lineFollowConfig → PID 参数  │
│  6. 渲染 Jinja2 模板                  │
│  7. 嵌入用户代码区域                  │
│  8. 输出完整 .py 文件内容              │
└──────────┬───────────────────────────┘
           │
           ▼
┌─────────────────────┐
│   生成的 Python 代码  │
│   (完整 .py 文件)     │
└──────────┬──────────┘
           │
           ▼
┌──────────────────────────────────────┐
│         文件下载（Browser）            │
│  Blob → URL.createObjectURL →        │
│  <a download="robot.py">点击下载      │
└──────────────────────────────────────┘
```

### 3.2 核心类设计（Python 后端）

```python
# code_generator.py

from jinja2 import Environment, FileSystemLoader, select_autoescape
from pathlib import Path
import json
from datetime import datetime


class CodeGenerator:
    def __init__(self, template_dir: str = "templates"):
        self.env = Environment(
            loader=FileSystemLoader(template_dir),
            autoescape=select_autoescape(['html', 'xml']),
            trim_blocks=True,
            lstrip_blocks=True,
        )
        self.env.filters['port_const'] = self._to_port_const

    def generate(self, vehicle_json: dict) -> str:
        """将 vehicles.json 配置转换为 Python 代码"""
        ctx = self._build_context(vehicle_json)
        template = self.env.get_template("base.py.jinja2")
        return template.render(**ctx)

    def _build_context(self, v: dict) -> dict:
        """从 vehicles.json 提取模板渲染所需的上下文"""
        hub_ports = v.get("hub", {}).get("ports", {})
        wheel_groups = v.get("wheelGroups", [])
        line_configs = v.get("lineFollowConfigs", [])

        # --- 设备检测 ---
        has_motor   = any(d.get("device") == "Motor" for d in hub_ports.values() if d)
        has_color   = any(d.get("device") == "ColorSensor" for d in hub_ports.values() if d)
        has_ultra   = any(d.get("device") == "UltrasonicSensor" for d in hub_ports.values() if d)

        # --- 端口常量映射 ---
        port_mappings = {}
        for port_letter, mapping in hub_ports.items():
            if mapping is None:
                continue
            device = mapping.get("device", "")
            suffix = {"Motor": "MOTOR", "ColorSensor": "SENSOR"}.get(device, "DEVICE")
            port_mappings[port_letter] = {
                "constant_name": f"{suffix}_PORT",
                "port": port_letter,
            }

        # --- 电机列表 ---
        motors = []
        for port_letter, mapping in hub_ports.items():
            if mapping and mapping.get("device") == "Motor":
                cfg = mapping.get("config", {})
                direction = "COUNTERCLOCKWISE" if cfg.get("reversed") else "CLOCKWISE"
                motors.append({
                    "var_name":    f"{port_letter.lower()}_motor",
                    "port_const":  f"{suffix}_PORT",
                    "direction":   direction,
                })

        # --- 轮组（DriveBase 参数） ---
        wg = wheel_groups[0] if wheel_groups else {}
        left_m  = wg.get("leftMotor", {})
        right_m = wg.get("rightMotor", {})

        # --- 巡线配置 ---
        lf = line_configs[0] if line_configs else {}

        return {
            "vehicle":         v,
            "timestamp":       datetime.now().isoformat(),
            "port_mappings":   port_mappings,
            "motors":          motors,
            "has_motor":       has_motor,
            "has_color_sensor": has_color,
            "has_ultrasonic":  has_ultra,
            "has_drivebase":   bool(wheel_groups),
            "has_line_follow": bool(line_configs),
            "left_motor":      {"var_name": f"{left_m.get('port','A').lower()}_motor"},
            "right_motor":     {"var_name": f"{right_m.get('port','B').lower()}_motor"},
            "wheel_diameter":  wg.get("wheelDiameter", 62),
            "axle_track":      wg.get("axleTrack", 112),
            "default_kp":      lf.get("kp", 1.5),
            "default_wait_ms": lf.get("scanIntervalMs", 10),
            "default_threshold": lf.get("threshold", 35),
        }

    @staticmethod
    def _to_port_const(port: str) -> str:
        return f"Port.{port.upper()}"
```

### 3.3 前端调用（JavaScript）

```javascript
// 调用后端 API 生成代码
async function generateCode(vehicleId) {
  const vehicles = JSON.parse(localStorage.getItem('vehicles') || '{"vehicles":[]}');
  const vehicle = vehicles.vehicles.find(v => v.id === vehicleId);
  if (!vehicle) return;

  const response = await fetch('/api/generate-code', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(vehicle),
  });

  const pythonCode = await response.text();
  showDownloadDialog(pythonCode, `${vehicle.name}.py`);
}

function showDownloadDialog(code, filename) {
  const blob = new Blob([code], { type: 'text/x-python' });
  const url  = URL.createObjectURL(blob);
  // 创建临时 <a> 触发下载
  const a = document.createElement('a');
  a.href     = url;
  a.download = filename;
  a.click();
  URL.revokeObjectURL(url);
}
```

## 四、代码生成内容详解

### 4.1 生成内容清单

| 区块 | 内容 | 来源字段 |
|---|---|---|
| **文件头注释** | 版本、生成时间、车辆名称 | `vehicle.name`, `timestamp` |
| **Import 语句** | 按需生成的 import | 遍历 `hub.ports` 检测设备类型 |
| **端口常量** | `LEFT_MOTOR_PORT = Port.A` 等 | `wheelGroups[].leftMotor.port`, `wheelGroups[].rightMotor.port`, `lineFollowConfigs[].sensorPort` |
| **全局变量** | `hub = None`, `left_motor = None` 等 | 根据检测到的设备生成 |
| **`init()` 函数** | Hub 初始化 + 所有设备实例化 | `hub.type`, `ports`, `wheelGroups` |
| **`calibrate()` 函数** | 白黑校准，返回阈值 | 始终生成（ColorSensor 存在时启用） |
| **`follow_line()` 函数** | PID 巡线逻辑 | `lineFollowConfigs[].kp`, `scanIntervalMs` |
| **`turn()` 函数** | 原地/弧线转向 | 始终生成（DriveBase 存在时） |
| **用户代码区域** | 保留手写代码 | 存储在 `programs.json` 中 |
| **入口调用** | `init()` + `threshold = calibrate()` | 始终生成 |

### 4.2 用户代码区域保留机制

**存储位置**：`programs.json`（独立于 vehicles.json）

```json
{
  "programs": [
    {
      "id": "uuid",
      "vehicleId": "vehicle-uuid",
      "name": "主程序",
      "userCode": "# === USER CODE START ===\nfollow_line(200, True, 950, 1.5, 10, True)\nturn(90, 0, 800, True)\n# === USER CODE END ==="
    }
  ]
}
```

**代码注入流程**：

```
vehicles.json ──┐
                ├──► 代码生成器
programs.json ──┘        │
                         ▼
              模板中 {{ user_code_initial }} 被替换
              → 用户代码原样嵌入 # === USER CODE START === / END === 之间
```

**模板中的标记**：

```python
# === USER CODE START ===
{{ user_code_initial }}
# === USER CODE END ===

# ... 后续 auto-gen 代码 ...
```

**覆盖规则**：
- **首次生成**：用户代码区域为空，生成注释提示
- **增量更新**：用户修改车辆配置 → 只更新非用户区域 → 保留用户代码不变
- **完全重写**：用户点击"重新生成" → 给出警告 → 清除用户区域重新开始

### 4.3 Dashboard Base 配置

当用户配置了轮组时，`DriveBase` 参数自动计算：

```python
drive_base = DriveBase(
    left_motor,                      # 左侧电机实例
    right_motor,                     # 右侧电机实例
    wheel_diameter={{ wheel_diameter }},   # 来自 wheelGroups[].wheelDiameter（单位：mm）
    axle_track={{ axle_track }}            # 来自 wheelGroups[].axleTrack（单位：mm）
)
```

**axle_track 估算**：
- 如果 UI 未提供，用户也未手动设置 → 使用默认值 `112`（SPIKE Prime 典型值）
- 精确值需要用户在地图页测量轮距后填入

### 5.2 输出的完整 .py 文件

```python
# ============================================================
# Auto-generated by Pybricks IDE — DO NOT EDIT MANUALLY
# Vehicle: 皮皮号
# Generated at: 2026-04-27T21:22:00+08:00
# ============================================================

from pybricks.hubs import PrimeHub
from pybricks.pupdevices import Motor, ColorSensor
from pybricks.parameters import Port, Direction, Stop
from pybricks.robotics import DriveBase
from pybricks.tools import wait

LEFT_MOTOR_PORT  = Port.A
RIGHT_MOTOR_PORT = Port.B
LINE_SENSOR_PORT = Port.E

MAX_MOTOR_SPEED = 1180

hub        = None
left_motor  = None
right_motor = None
line_sensor = None
drive_base  = None


def init():
    global hub, left_motor, right_motor, line_sensor, drive_base

    hub = PrimeHub()
    left_motor  = Motor(LEFT_MOTOR_PORT,  Direction.COUNTERCLOCKWISE)
    right_motor = Motor(RIGHT_MOTOR_PORT, Direction.CLOCKWISE)
    line_sensor = ColorSensor(LINE_SENSOR_PORT)
    drive_base = DriveBase(
        left_motor,
        right_motor,
        wheel_diameter=62,
        axle_track=112
    )
    while not hub.imu.ready():
        wait(10)


def calibrate():
    white_sum = 0
    for _ in range(10):
        white_sum += line_sensor.reflection()
        wait(20)
    white_ref = white_sum // 10
    wait(500)
    black_sum = 0
    for _ in range(10):
        black_sum += line_sensor.reflection()
        wait(20)
    black_ref = black_sum // 10
    return (white_ref + black_ref) // 2


def follow_line(distance, enable_lf, base_speed, kp=1.5, wait_ms=10, stop_on_end=True):
    base = base_speed
    max_s = int(MAX_MOTOR_SPEED * 0.9)
    if base > max_s:
        base = max_s
    min_s = int(MAX_MOTOR_SPEED * -0.3)
    drive_base.reset()
    while abs(drive_base.distance()) < distance:
        if enable_lf:
            reflection = line_sensor.reflection()
            error = reflection - threshold
            turn_rate = kp * error
            left_speed  = base - turn_rate
            right_speed = base + turn_rate
            left_speed  = max(min_s, min(max_s, left_speed))
            right_speed = max(min_s, min(max_s, right_speed))
            left_motor.run(left_speed)
            right_motor.run(right_speed)
        else:
            drive_base.drive(base, 0)
        wait(wait_ms)
    if stop_on_end:
        left_motor.stop()
        right_motor.stop()


def turn(angle, radius, speed, stop_on_end=True):
    max_s = int(MAX_MOTOR_SPEED * 0.9)
    if speed > max_s:
        speed = max_s
    if radius == 0:
        drive_base.turn(angle, then=Stop.HOLD if stop_on_end else Stop.COAST)
    else:
        drive_base.curve(radius, angle, then=Stop.HOLD if stop_on_end else Stop.COAST)
    if stop_on_end:
        wait(100)
```

## 六、文件下载

### 6.1 纯前端方案（无需后端）

适用于：所有逻辑在浏览器本地完成（Web IDE），代码生成也在前端用 JavaScript 渲染 Jinja2。

```javascript
// 前端模板渲染（使用 jinja2-js 或手写简易模板）
function renderTemplate(templateStr, context) {
  return templateStr.replace(/\{\{(\w+)\}\}/g, (_, key) => context[key] ?? '');
}

async function downloadPythonFile(vehicleJson) {
  const templateEl = document.getElementById('code-template');
  const template   = templateEl.textContent;
  const context    = buildContext(vehicleJson);
  const code       = renderTemplate(template, context);

  const blob = new Blob([code], { type: 'text/x-python' });
  const url  = URL.createObjectURL(blob);

  const a = document.createElement('a');
  a.href     = url;
  a.download = `${vehicleJson.name || 'robot'}.py`;
  document.body.appendChild(a);
  a.click();
  document.body.removeChild(a);
  URL.revokeObjectURL(url);
}
```

### 6.3 预览 + 下载双模式

**预览模式**：在网页 `<pre><code>` 区域显示生成代码（只读，不可下载前修改）

**下载模式**：点击下载按钮，Blob → createObjectURL → 触发下载

```javascript
function previewCode(code) {
  const pre = document.getElementById('code-preview');
  pre.textContent = code;
  hljs.highlightElement(pre);  // 语法高亮
}

function downloadCode(code, filename) {
  const blob = new Blob([code], { type: 'text/x-python' });
  if (window.downloadCounter === undefined) window.downloadCounter = 0;
  window.downloadCounter++;

  const url = URL.createObjectURL(blob);
  const link = document.createElement('a');
  link.href     = url;
  link.download = filename;
  link.click();
  // 延迟回收，避免部分浏览器立即回收导致下载失败
  setTimeout(() => URL.revokeObjectURL(url), 10000);
}
```

### 6.4 直接复制到剪贴板（轻量化替代）

```javascript
async function copyToClipboard(code) {
  try {
    await navigator.clipboard.writeText(code);
    showToast('代码已复制到剪贴板');
  } catch {
    // Fallback
    const ta = document.createElement('textarea');
    ta.value = code;
    document.body.appendChild(ta);
    ta.select();
    document.execCommand('copy');
    document.body.removeChild(ta);
    showToast('代码已复制到剪贴板');
  }
}
```

## 七、关键技术决策总结

| 决策点 | 选择 | 理由 |
|---|---|---|
| **模板引擎** | jinja2 | 功能完整，条件/循环/宏支持好，生态成熟 |
| **代码生成时机** | 用户点击"生成"时实时生成 | 避免每次保存都触发重写，用户可控 |
| **用户代码保留** | 独立 `programs.json` 存储，模板标记区域注入 | 变更配置不影响用户手写代码 |
| **DriveBase 参数** | 从 `wheelGroups[].wheelDiameter` + `axleTrack` 读取 | 直接对应物理参数，无需额外计算 |
| **巡线 PID** | `kp` 来自 UI 配置，`ki/kd` 暂固定为 0 | 单 P 控制足够简单场景，KI/KD 供高级用户手动添加 |
| **下载方式** | Blob + createObjectURL（纯前端） | 无需后端，用户即刻拿到 .py 文件 |
| **文件编码** | UTF-8 | 支持中文注释和变量名 |
